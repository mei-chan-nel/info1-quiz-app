QUESTION 011#center_style_fe_curated_015

1件2,000バイトの伝票データを2件ずつまとめ、400バイトのヘッダを付けて送る。伝票が1時間に100,000件発生し、回線速度が1Mビット/秒のとき、回線利用率はおよそ何%か。 ただし、1Mビット＝1,000,000ビットとする。

  1. 0

    6.1

  2. 1

    44

  3. 2

    49

  4. 3

    53

正答と解説を確認

正答2. 49

解説

2件ごとに送るので送信回数は50,000回/時である。1回のデータ量は2,000×2+400=4,400バイト、合計220,000,000バイト=1,760,000,000ビットである。1Mビット/秒の1時間分は3,600,000,000ビットなので、約49%。

QUESTION 013#center_style_fe_curated_023

64kビット/秒の回線で、10⁶バイトのファイルを送信する。伝送効率が80%のとき、伝送におよそ何秒かかるか。 ただし、1kビット＝1,000ビットとする。

  1. 0

    19.6

  2. 1

    100

  3. 2

    125

  4. 3

    156

正答と解説を確認

正答3. 156

解説

実効速度は64k×0.8＝51.2kビット/秒である。10⁶バイトは8×10⁶ビット＝8,000kビットなので、8,000÷51.2＝156.25秒となる。

QUESTION 016#center_style_ap_curated_044

伝送速度64kビット/秒の回線でデータを連続送信したところ、平均して100秒に1回、1ビットの誤りが発生した。この回線のビット誤り率として最も近いものはどれか。 ただし、1kビット＝1,000ビットとする。

  1. 0

    1.95×10⁻⁸

  2. 1

    1.56×10⁻⁷

  3. 2

    1.95×10⁻⁵

  4. 3

    1.56×10⁻⁴

正答と解説を確認

正答1. 1.56×10⁻⁷

解説

100秒間に送るビット数は64,000×100＝6,400,000ビットである。1ビットの誤りなので、誤り率は1÷6,400,000＝1.5625×10⁻⁷となる。

QUESTION 017#center_style_ap_curated_057

LANの速度が100Mビット/秒、伝送効率が80%で、インターネット側の実効速度は90Mビット/秒である。540Mバイトのファイルをダウンロードするとき、かかる時間はおよそ何秒か。ここで、その他の遅延は考えない。 ただし、1Mビット＝1,000,000ビット、1Mバイト＝1,000,000バイトとする。

  1. 0

    43

  2. 1

    48

  3. 2

    54

  4. 3

    60

正答と解説を確認

正答2. 54

解説

LANの実効速度は100×0.8=80Mビット/秒で，インターネット側90Mビット/秒より遅いので，実際には80Mビット/秒で考える。540Mバイトは4,320Mビットなので，4,320÷80=54秒である。
