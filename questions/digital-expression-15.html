QUESTION 141#original_knowledge_0025

16進数 6D₁₆ を8ビットの2進数で表したものはどれか。

  1. 0

    01101101₂

  2. 1

    01101011₂

  3. 2

    11010110₂

  4. 3

    00001101₂

正答と解説を確認

正答0. 01101101₂

解説

16進数の6は0110、Dは1101なので、連結して01101101₂となる。 16進数は16種類の記号(0,1,2,…,9,A,B,C,D,E,F)を用いて数を表し、16のべきで位が上がり、2進数は2種類の記号(0,1)を用いて数を表し、2のべきで位が上がる。 16進数→（慣れている）10進数→2進数と変換する。 10進数ー16進数ー2進数の表を見て、下から4桁ずつに変換するとよい。

QUESTION 142#original_knowledge_0026

4ビットの2の補数表現で 0101₂ の符号を反転した値はどれか。

  1. 0

    1010₂

  2. 1

    1011₂

  3. 2

    0110₂

  4. 3

    0101₂

正答と解説を確認

正答1. 1011₂

解説

全ビットを反転して1010₂とし、1を加えると1011₂になる。0101₂+1011₂は下位4ビットで0000₂となる。 2進数の場合、元の数の0と1を入れ替えて、それに1を加えた数である。 実際、0101(2)+1011(2)＝10000(2)＝0000(2)である。 4ビット表記(固定長)において、0101(2)+1011(2)＝0000(2)であったので、1011(2)＝－5と考えられる。 4ビット表記、符号付きの数は以下のようにまとめられる（表）。 これにより、固定長なら足し算だけで引き算を計算することが可能になったと言うこともできる。

QUESTION 143#original_knowledge_0027

一般的なPython実装で0.1 + 0.2 == 0.3がFalseになる主な理由はどれか。

  1. 0

    Pythonでは小数の加算が禁止されているため。

  2. 1

    ==が数値を文字列として比較するため。

  3. 2

    0.1などを2進浮動小数点で正確に表せないため。

  4. 3

    0.1+0.2の数学的な答えが0.3でないため。

正答と解説を確認

正答2. 0.1などを2進浮動小数点で正確に表せないため。

解説

0.1や0.2は2進数では有限小数にならず、近似値として保持されるため丸め誤差が生じる。 0.1+0.2を計算する際、浮動小数点数による丸め誤差が発生するため、0.3と等しくならない。 よって、0.1＝1.1001 1001 1001 …×2⁻⁴である。 0.2＝1.1001 1001 1001 …×2⁻³である。 Pythonでは通常、倍精度浮動小数点数（符号部1ビット、指数部11ビット、仮数部52ビット：全体で64ビット）で数を表現し、計算している。 仮数部を52桁になるように（四捨五入のような計算で）丸められる。 これにより誤差が生じる。 0.1と0.2は2進数では循環する小数になるため、浮動小数点数にすると丸め誤差が生じる。 浮動小数点数を用いて数を丸めたもの同士を足し合わせるため、0.3ちょうどにはならない。 0.30000000000000004となる。

QUESTION 144#original_knowledge_0028

同じ音をデジタル化するとき、一般に元の波形をより細かく表現できる設定はどれか。

  1. 0

    標本化周期を長くし、量子化ビット数を小さくする。

  2. 1

    標本化周波数を低くし、量子化ビット数だけを0にする。

  3. 2

    標本化周期と標本化周波数を同時に大きくする。

  4. 3

    標本化周波数を高くし、量子化ビット数を大きくする。

正答と解説を確認

正答3. 標本化周波数を高くし、量子化ビット数を大きくする。

解説

標本化周波数を高くすると時間方向に細かく測定でき、量子化ビット数を大きくすると振幅方向の段階を細かくできる。そのため、どちらも大きくする設定が、一般に元の波形をより細かく表現できる。なお標本化定理では、帯域制限された信号を理想的に復元するには、信号に含まれる最高周波数の2倍を超える標本化周波数が必要とされる。実際のディジタル化では量子化誤差なども生じるため、標本化周波数だけで完全な再現が保証されるわけではない。

QUESTION 145#original_knowledge_0029

1200×800ピクセル、1ピクセル当たり24ビットの無圧縮画像のデータ量は約何MBか。1MB=1,000,000Bとし、付加情報は無視する。

  1. 0

    2.88MB

  2. 1

    0.96MB

  3. 2

    23.04MB

  4. 3

    28.8MB

正答と解説を確認

正答0. 2.88MB

解説

1200×800×24÷8=2,880,000B=2.88MBである。 画像は小さな単色の点（ピクセル）の集まりである。 この画像は、1200×800＝960,000ピクセルである。 24ビットフルカラーとは、各ピクセルに色情報を24ビット分付与しているという意味である。 具体的には、RGB（赤・緑・青＝光の三原色）それぞれに8ビットずつ、2⁸＝256段階の色を表現している。 全体では2⁸×2⁸×2⁸＝2²⁴＝約1,677万色を表現できる。 8ビットは1バイトであるので、24ビットは3バイトである。

QUESTION 146#original_knowledge_0030

標本化周波数44.1kHz、量子化16ビット、5分間のモノラル音声を無圧縮で保存する。データ量に最も近いものはどれか。1MB=1,000,000Bとする。

  1. 0

    約5.3MB

  2. 1

    約26.5MB

  3. 2

    約52.9MB

  4. 3

    約211.7MB

正答と解説を確認

正答1. 約26.5MB

解説

44,100回/秒×16ビット×300秒÷8=26,460,000Bなので約26.5MBである。 標本化周波数44.1kHzは、1秒間に44,100回、音の振幅を測ることを表す。5分は300秒なので標本数は44,100×300=13,230,000個。1標本は16ビット=2バイトで、モノラルは1チャンネルだから、13,230,000×2=26,460,000Bとなる。1MB=1,000,000Bより26.46MB、選択肢では約26.5MBが最も近い。ステレオなら2チャンネルなので、この2倍になる。

QUESTION 147#original_knowledge_0031

平均ビットレート5Mbpsの動画を5分間保存したとき、データ量は何MBか。1MB=1,000,000Bとする。

  1. 0

    37.5MB

  2. 1

    625MB

  3. 2

    187.5MB

  4. 3

    1,500MB

正答と解説を確認

正答2. 187.5MB

解説

5,000,000ビット/秒×300秒÷8=187,500,000B=187.5MBである。 ビットレートはデータを転送したり、記録するときの速度を表す。 bps（ビット パー セカンド）は1秒間で何ビットのデータ量を転送・記録するかを表している。 5Mbpsは1秒間で5メガビットのデータ量という意味である。 5分＝300秒でのデータ量は300(秒)×5(メガビット/秒)＝1,500メガビットである。 8ビット＝1バイトなので8メガビット＝1MBである。 よって、1,500(メガビット)÷8＝187.5MBである。

QUESTION 148#original_knowledge_0032

可逆圧縮の特徴として、最も適切なものはどれか。

  1. 0

    展開しても元のデータを完全には復元できない。

  2. 1

    画像や音声の画質を必ず低下させる。

  3. 2

    MP3だけに用いられる音声専用の方式である。

  4. 3

    展開すると元のデータを完全に復元できる。

正答と解説を確認

正答3. 展開すると元のデータを完全に復元できる。

解説

可逆圧縮は、展開したときに元のデータを完全に復元できる圧縮方式である。文書やプログラムなど、1ビットでも失うと内容が変わり得るデータに適している。これに対して非可逆圧縮は、知覚しにくい情報などを取り除いてデータ量を減らすため、元のデータを完全には復元できない。MP3は代表的な非可逆音声圧縮方式であり、可逆圧縮ではない。

QUESTION 149#original_knowledge_0033

学校のロゴを、名刺から大型看板まで輪郭を滑らかに保って使いたい。適した画像形式の考え方はどれか。

  1. 0

    図形を座標や数式で表すベクタ画像として作成する。

  2. 1

    低解像度のラスタ画像だけを保存し、拡大時は各ピクセルを大きくする。

  3. 2

    色数を1色にすれば、ラスタ画像でも解像度に関係なく滑らかになる。

  4. 3

    音声用の非可逆圧縮を使って輪郭情報を保存する。

正答と解説を確認

正答0. 図形を座標や数式で表すベクタ画像として作成する。

解説

ベクタ画像は拡大・縮小しても輪郭が劣化しにくく、ロゴや図形に適する。 ラスタ画像（ビットマップ画像）はピクセルと呼ばれる小さな点を格子状に配置して画像を表現する。 写真のように色やグラデーションが複雑な画像の表現に向いている。 一方で、格子状の点の集まりのため斜め線がギザギザになったり、拡大で画像が粗く劣化する。 ベクタ画像は線分や円、多角形などの図形の集合として画像を表現する。 拡大や縮小、回転などが自由に行える。

QUESTION 150#original_knowledge_0034

2入力A、Bが異なるときだけ1を出力する論理演算はどれか。

  1. 0

    AND（論理積）

  2. 1

    XOR（排他的論理和）

  3. 2

    OR（論理和）

  4. 3

    NOT（否定）

正答と解説を確認

正答1. XOR（排他的論理和）

解説

XORは00と11で0、01と10で1になる。 XOR（排他的論理和）は、AとBの一方だけが1のときに1を出す。真理値表は00→0、01→1、10→1、11→0となる。ANDは11だけが1、ORは01・10・11が1なので、いずれもXORとは異なる。AND・OR・NOTだけで表すなら `(A AND NOT B) OR (NOT A AND B)` となる。
