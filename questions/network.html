QUESTION 003#center_style_common_test_0032

送信者が kaoru@example.ac.jp，正しい受信者が shinobu@example.ed.jp である。 受信者のアドレスを誤って sinobu@example.ed.jp と入力した。example.ed.jp というドメインは存在するが，sinobu というユーザ名は存在しない。 このメールを配信できないことを最初に検出するものを，次の選択肢から一つ選べ。

  1. 0

    送信側メールサーバが，送信者のドメインが存在しないと検出する。

  2. 1

    送信側メールサーバが，受信者のドメインが存在しないと検出する。

  3. 2

    受信側メールサーバが，受信者のユーザ名が存在しないと検出する。

  4. 3

    受信側メールサーバが，送信者のユーザ名が存在しないと検出する。

正答と解説を確認

正答2. 受信側メールサーバが，受信者のユーザ名が存在しないと検出する。

解説

メールアドレスでは，@の後ろがドメイン名，@の前がそのドメイン内のユーザ名である。example.ed.jp は存在するので送信側は受信側サーバを見つけられるが，sinobu というユーザ名は受信側サーバ内に存在しないため，受信側メールサーバが最初に検出する。

QUESTION 004#center_style_common_test_0033

  1. 0

    送信側メールサーバが，受信側のドメイン名が存在しないと検出する。

  2. 1

    受信側メールサーバが，受信側のドメイン名が存在しないと検出する。

  3. 2

    送信側メールサーバが，受信者のユーザ名が存在しないと検出する。

  4. 3

QUESTION 009#center_style_fe_curated_005

符号化速度192kビット/秒の音声データ2.4Mバイトを、通信速度128kビット/秒でダウンロードしながら途切れずに再生したい。再生前にあらかじめ蓄えておく必要があるデータの再生時間は最低何秒か。ただし、1kビット＝1,000ビット、1Mバイト＝1,000,000バイトとする。

  1. 0

    50

  2. 1

    100

  3. 2

    150

  4. 3

    250

正答と解説を確認

正答0. 50

解説

2.4Mバイトは19.2Mビットで、192kビット/秒の音声なら再生時間は100秒である。再生中は1秒ごとに192kビット必要だが128kビットしか届かないので、差の64kビット/秒を100秒分、つまり6.4Mビット先に蓄える必要がある。128kビット/秒で6.4Mビットを受け取る時間は50秒である。

QUESTION 010#center_style_fe_curated_014

10Mビット/秒の回線で、平均1Mバイトのファイルを10秒ごとに転送する。転送量の20%が制御情報として付加されるとき、回線利用率は何%か。 ただし、1Mビット＝1,000,000ビット、1Mバイト＝1,000,000バイトとする。

  1. 0

    1.2

  2. 1

    6.4

  3. 2

    8.0

  4. 3

    9.6

正答と解説を確認

正答3. 9.6

解説

1Mバイトは8Mビットで、制御情報20%を加えると9.6Mビットである。これを10秒ごとに送るので平均0.96Mビット/秒、10Mビット/秒に対して9.6%である。
