QUESTION 072#center_style_ap_curated_115

サンプリング周波数40kHz，量子化ビット数16ビットでA-D変換したモノラル音声がある。1秒間のデータ量は何kバイトか。ここで，1kバイトは1,000バイトとする。

  1. 0

    80

  2. 1

    20

  3. 2

    40

  4. 3

    640

正答と解説を確認

正答0. 80

解説

1秒間に40,000回標本化し，各標本が16ビットなので，40,000×16=640,000ビットである。8で割ると80,000バイト，すなわち80kバイトである。

QUESTION 074#center_style_ap_curated_117

nを2以上の整数とする。正の整数aを2進数で表すとnビットであり，b＝a²とする。aが取り得る範囲で，bを2進数で表すために必要となるビット数の最大値はどれか。

  1. 0

    2n

  2. 1

    n+1

  3. 2

    n

  4. 3

    3n

正答と解説を確認

正答0. 2n

解説

nビットの正の整数aが取り得る最大値は2ⁿ－1である。したがってbの最大値は（2ⁿ－1）²となる。n≧2では，2⁽²ⁿ⁻¹⁾≦（2ⁿ－1）²＜2²ⁿが成り立つため，この値を2進数で表すには2nビットが必要である。よって，必要となるビット数の最大値は2nビットである。

QUESTION 075#center_style_ap_curated_119

100Mビット/秒のLANを使用し，1件1,000バイトのデータを1,000件連続して送る。LANの伝送効率が50%のとき，伝送に必要な時間は何秒か。 ただし、1Mビット＝1,000,000ビットとする。

  1. 0

    0.16

  2. 1

    0.02

  3. 2

    0.08

  4. 3

    1.6

正答と解説を確認

正答0. 0.16

解説

送るデータ量は1,000バイト×1,000件=1,000,000バイト=8,000,000ビットである。実効速度は100Mビット/秒×50%=50,000,000ビット/秒なので，8,000,000÷50,000,000=0.16秒である。

QUESTION 076#center_style_ip_curated_211

音を同じ標本化周波数で記録するとき，量子化ビット数を8ビットから16ビットに増やした場合の説明として最も適当なものはどれか。

  1. 0

    音の振幅をより細かい段階で記録できる。

  2. 1

    同じ記録時間なら，データ量は少なくなる。

  3. 2

    記録できる周波数の上限が高くなる。

    記録したデータの処理に必要な計算量は必ず減る。

QUESTION 077#center_style_ip_curated_212

RGBの各色の階調をそれぞれ3ビットで表す。混色で表せる色は全部で何通りか。

  1. 0

    512通り

  2. 1

    24通り

  3. 2

    256通り

  4. 3

    4,096通り

正答と解説を確認

正答0. 512通り

解説

3ビットでは1色あたり2³＝8段階を表せる。RGBの3色を組み合わせるので，8×8×8=512通りである。

QUESTION 080#center_style_fe_curated_112

9ビットのD-A変換器で、入力値が1増えると出力電圧が5mV増える。入力値0の出力を0mVとすると、2進数110000000を入力したときの出力電圧は何Vか。

  1. 0

    0.96V

  2. 1

    1.28V

  3. 2

    1.92V

  4. 3

    2.56V

正答と解説を確認

正答2. 1.92V

解説

2進数110000000は、2⁸＋2⁷＝256＋128＝384である。入力値1当たり5mVなので、384×5＝1,920mV＝1.92Vとなる。
