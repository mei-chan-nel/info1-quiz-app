QUESTION 031#center_style_ip_curated_093

3桁の社員コードで、上位2桁の各数字を足した値の1の位をチェックディジットとして3桁目に付ける。上位2桁が57のとき、チェックディジットとして最も適当なものはどれか。

  1. 0

    1

  2. 1

    2

  3. 2

    5

  4. 3

    7

正答と解説を確認

正答1. 2

解説

上位2桁57では、5+7=12である。1の位は2なのでチェックディジットは2になる。

QUESTION 032#center_style_ap_curated_079

ある店では、翌週の販売予測量を先週と今週の販売量の平均とし、安全在庫量を販売予測量の10%とする。先週の販売量が80個、今週が100個、今週末の在庫量が35個のとき、「販売予測量＋安全在庫量－現在の在庫量」で求める補充量は何個か。

  1. 0

    55個

  2. 1

    64個

  3. 2

    90個

  4. 3

    99個

正答と解説を確認

正答1. 64個

解説

翌週の販売予測量は（80＋100）÷2＝90個、安全在庫量は90×0.1＝9個である。したがって補充量は90＋9－35＝64個となる。

QUESTION 034#center_style_fe_curated_065

4桁のデータ4816にチェックディジットを付ける。各桁に左から順に重み2、1、2、1を掛け、その和を10で割った余りをrとする。（10－r）を10で割った余りをチェックディジットとすると、付ける数字はどれか。

  1. 0

    0

  2. 1

    4

  3. 2

    6

  4. 3

    8

正答と解説を確認

正答2. 6

解説

重み付きの和は4×2＋8×1＋1×2＋6×1＝24で、10で割った余りrは4である。（10－4）を10で割った余りは6なので、チェックディジットは6となる。

QUESTION 037#center_style_ap_curated_084

小文字のアルファベット a から z に，順に 0 から 25 を対応させる。ハッシュ値を「対応する数を10で割った余り」とすると，同じハッシュ値になる組合せはどれか。

  1. 0

    a と i

  2. 1

    b と m

  3. 2

    c と l

  4. 3

    d と n

正答と解説を確認

正答3. d と n

解説

dは3，nは13に対応し，どちらも10で割った余りは3なので同じハッシュ値になる。ハッシュ関数の規則をそれぞれの文字に適用して，同じ余りになる組合せを探す。
