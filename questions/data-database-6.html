QUESTION 051#center_style_ip_curated_236

関係データベースで，二つの表に共通する値がある行を組み合わせて，新しい表を作る操作として最も適当なものはどれか。

  1. 0

    条件に合う行だけを取り出す。

  2. 1

    必要な列だけを取り出す。

  3. 2

    共通する値をもつ行どうしを組み合わせる。

  4. 3

    表のすべてのデータを暗号化する。

正答と解説を確認

正答2. 共通する値をもつ行どうしを組み合わせる。

解説

結合は，二つの表の共通する値を手掛かりに，対応する行どうしを組み合わせる操作である。

QUESTION 054#center_style_curated_068_003

AIを活用している例として，最も適当なものはどれか。

  1. 0

    大量の道路画像を学習したモデルが，車載カメラの画像から道路標識を分類する。

  2. 1

    あらかじめ指定した時刻になると，タイマーが照明を点灯する。

  3. 2

    表計算ソフトが，入力された数値を設定済みの式で合計する。

  4. 3

    温度が設定値を超えると，センサと制御回路が換気扇を動かす。

QUESTION 057#center_style_curated_075_002

データベースの表で，各行を一意に識別するために使う列または列の組の説明として，最も適当なものはどれか。

  1. 0

    値を空欄にできない列であれば，必ず行を一意に識別できる

  2. 1

    検索を速くするために，データの格納位置を記録する

  3. 2

    異なる表の列と必ず関連付けられている

  4. 3

    サーバをクラウドに移し，新しいサーバを短時間で用意できるようにした。

  2. 1

    自動車から集めた位置情報や急ブレーキの情報を分析し，危険な地点を知らせた。
