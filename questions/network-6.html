QUESTION 051#center_style_curated_066_003

DoS攻撃の説明として，最も適当なものはどれか。

  1. 0

    他人になりすましてネットワーク上のサービスを利用すること

  2. 1

    通信経路上で他人のデータを盗み見ること

  3. 2

    大量の要求を送り続け，サービスを利用しにくくすること

  4. 3

    文字の組合せを順に試し，パスワードを推測すること

正答と解説を確認

正答2. 大量の要求を送り続け，サービスを利用しにくくすること

解説

DoS攻撃は，大量の通信や要求を送るなどしてサーバやネットワークに負荷をかけ，サービスを利用しにくくする攻撃である。

QUESTION 054#center_style_curated_066_017

異なるメーカーのセンサと制御装置を接続したところ，送るデータの形式や応答の順序が一致せず通信できなかった。相互に通信できるようにする対応として，最も適当なものはどれか。

  1. 0

    両方の機器が扱う信号をアナログ信号に変更する。

  2. 1

    同じメーカーの機器へ交換し，通信仕様は確認しない。

  3. 2

    通信速度だけを高くし，データ形式や応答順序は変更しない。

  4. 3

    両方の機器を，同じ通信プロトコルの仕様に従うよう設定する。

正答と解説を確認

正答3. 両方の機器を，同じ通信プロトコルの仕様に従うよう設定する。

解説

通信プロトコルは，通信時のデータ形式，送受信の順序，エラー時の処理などに関する取り決めである。メーカーが異なる機器同士であっても，同じ通信プロトコルの仕様に従えば，相互に通信できる。

QUESTION 056#center_style_curated_067_002

ブラウザでURLが https:// で始まるWebページを閲覧するとき，通信の暗号化について最も適当なものはどれか。

  1. 0

    利用者からWebサーバへ送る通信だけが暗号化される。

  2. 1

    Webサーバから利用者へ返る通信だけが暗号化される。

  3. 2

    利用者とWebサーバの双方向の通信が暗号化される。

  4. 3

    暗号化される方向は，利用者の画面の大きさで決まる。

QUESTION 057#center_style_curated_067_008

WANを利用する必要がある例として，最も適当なものはどれか。

  1. 0

    同じ教室内のPCとプリンタを接続する。

  2. 1

    同じ部屋のPC同士でファイルを共有する。

  3. 2

    同じ建物内のPCから校内サーバを使う。

  4. 3

    遠く離れた本部と支部のネットワークを接続する。

正答と解説を確認

正答3. 遠く離れた本部と支部のネットワークを接続する。

QUESTION 059#center_style_curated_067_010

複数の顧客に同じ内容のメールを送るとき，他の顧客のメールアドレスを互いに見せないための宛先欄として，最も適当なものはどれか。

  1. 0

    To欄に全員を入れる。

  2. 1

    Cc欄に全員を入れる。

  3. 2

    BCC欄に顧客を入れる。

  4. 3

    件名欄に全員のメールアドレスを書く。

正答と解説を確認

正答2. BCC欄に顧客を入れる。

解説

BCC欄に入れた宛先は，他の受信者から見えない。複数の相手のメールアドレスを互いに見せたくない場合に使う。
