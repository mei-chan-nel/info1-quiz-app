QUESTION 061#center_style_curated_073_023

1対1の情報伝達経路が必要なチームで，人数が10人から15人に増えた。必要な経路は幾つ増えるか。

  1. 0

    5

  2. 1

    10

  3. 2

    60

  4. 3

    165

正答と解説を確認

正答2. 60

解説

10人では10×9÷2=45経路，15人では15×14÷2=105経路である。増加分は105-45=60経路である。

QUESTION 066#center_style_curated_083_003

変数AとBの値を入れ替える。値を一時的に保存する変数TMPを使う手順として，最も適当なものはどれか。

    TMP ← A，A ← B，B ← TMP

  2. 1

    TMP ← A，B ← TMP，A ← B

  3. 2

    A ← B，B ← A，TMP ← A

  4. 3

    TMP ← B，A ← TMP，B ← A

QUESTION 069#center_style_curated_085_016

ペンを下ろし，次の処理を繰り返して1辺1mの正六角形を描くプログラムを考える。 「1m直進する。右にA度向きを変える。これをB回行う。」 A，Bの組合せとして，最も適当なものはどれか。

  1. 0

    A=60，B=6

  2. 1

    A=90，B=4

  3. 2

    A=120，B=3

  4. 3

    A=45，B=8

正答と解説を確認

正答0. A=60，B=6

解説

正六角形は6本の辺から成り，外角は360度÷6=60度である。したがって，1m直進して右に60度曲がる処理を6回繰り返せばよい。

QUESTION 070#simple_original_097

Python、JavaScript、Cの共通点として最も適切なものはどれか。

QUESTION 071#simple_original_113

プログラムの基本的な実行の流れを構成する三つの制御構造はどれか。

  1. 0

    入力・出力・記憶

  2. 1

    加算・減算・乗算

  3. 2

    順次・分岐・反復

  4. 3

    検索・整列・圧縮

正答と解説を確認

正答2. 順次・分岐・反復

解説

構造化プログラミングでは、処理を順に行う順次、条件で選ぶ分岐、繰り返す反復が基本となる。 構造化プログラミングの基本は、順に実行する順次、条件で選ぶ分岐、繰り返す反復である。入出力や演算は処理内容であり、制御の構造そのものではない。

QUESTION 072#simple_original_116

自作の天気アプリが、外部サービスへリクエストを送り、予報データを決められた形式で受け取る仕組みはどれか。

  1. 0

    アルゴリズム

  2. 1

    コンパイラ

  3. 2

    デバッグ

  4. 3

    API

正答と解説を確認

正答3. API

解説

APIは、プログラムから他のソフトウェアやサービスの機能・データを利用するための定められた窓口である。 APIは、外部サービスの機能やデータを決められた手順で利用するための窓口である。アルゴリズムは手順、コンパイラは翻訳ツール、デバッグは修正作業を指す。

QUESTION 074#simple_original_132

数理モデルの例として最も適切なものはどれか。

  1. 0

    建物の形状を縮尺模型で表す。

  2. 1

    人口の増減を、出生率・死亡率・移動率を含む数式で表す。

  3. 2

    駅と乗換えの関係を路線図で表す。

  4. 3

    製品の外観を実物大の模型で表す。

正答と解説を確認

正答1. 人口の増減を、出生率・死亡率・移動率を含む数式で表す。

数理モデルは、対象の関係や変化を数式で表すモデルである。出生率などを用いて人口変化を式で表す例が該当する。 数式を用いることで、条件を変えたときの人口の変化を計算・予測できる。縮尺模型は物理モデル、路線図は関係を簡略化した図的なモデルである。
