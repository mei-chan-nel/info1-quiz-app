QUESTION 031#center_style_ap_curated_049

第n週の販売量をC[n]、第n週末の在庫量をB[n]とする。翌週の販売予測量を(C[n-1]+C[n])/2、安全在庫量を販売予測量の10%とすると、第n週末の補充量を表す式として最も適当なものはどれか。

  1. 0

    (C[n-1]+C[n])/2×1.1 - B[n]

  2. 1

    (C[n-1]+C[n])/2×1.1 - B[n-1]

  3. 2

    (C[n-1]+C[n])/2 + C[n]×0.1 - B[n]

  4. 3

    (C[n-2]+C[n-1])/2 + C[n]×0.1 - B[n]

正答と解説を確認

正答0. (C[n-1]+C[n])/2×1.1 - B[n]

解説

補充量は、翌週の販売予測量に安全在庫量を加え、現在の在庫量を引く。販売予測量をP=(C[n-1]+C[n])/2とすると、安全在庫量は0.1Pなので、P+0.1P-B[n]=P×1.1-B[n]である。

QUESTION 042#center_style_ap_curated_081

8ビットの2の補数表現で表せる整数の範囲として最も適当なものはどれか。

    -255から255

  2. 1

    -128から127

  3. 2

    -127から128

  4. 3

    0から255

正答と解説を確認

正答1. -128から127

解説

nビットの2の補数表現では，表せる整数の範囲は -2^(n-1) から 2^(n-1)-1 である。8ビットでは -128 から 127 になる。符号なし整数の範囲と混同しないようにする。
