QUESTION 081#center_style_ip_curated_146

有名な選手の写真を無断で商品に使って販売する行為によって，その選手の氏名や肖像がもつ顧客を引き付ける力を利用する権利を侵害するおそれがある。この権利として，最も適当なものはどれか。

  1. 0

    パブリシティの権利

  2. 1

    意匠権

  3. 2

    商標権

  4. 3

    特許権

正答と解説を確認

正答0. パブリシティの権利

解説

パブリシティの権利は，有名人などの氏名や肖像がもつ顧客吸引力を無断で商業利用されない利益に関係する。

QUESTION 088#center_style_ip_curated_163

パスワード管理として，最も適当なものはどれか。

  1. 0

    学校や仕事で使う重要なパスワードを，私的なサービスでは使い回さない。

  2. 1

    初期パスワードは覚えやすいので，できるだけ変更しない。

  3. 2

    数個のパスワードを全てのサービスで順番に使い回す。

  4. 3

    パスワードを平文のファイルに保存し，共有フォルダに置く。

正答と解説を確認

正答0. 学校や仕事で使う重要なパスワードを，私的なサービスでは使い回さない。

解説

重要なパスワードを他サービスで使い回すと，一つのサービスから漏えいしたときに被害が広がるため，使い回さないことが望ましい。

QUESTION 090#center_style_ip_curated_171

AさんがBさんだけに読ませたい電子メールを，公開鍵暗号方式でBさんの公開鍵を使って暗号化した。このメールを復号するために必要な鍵として，最も適当なものはどれか。

  1. 0

    Bさんの秘密鍵

  2. 1

    Bさんの公開鍵

  3. 2

    Aさんの公開鍵

  4. 3

    Aさんの秘密鍵

正答と解説を確認

正答0. Bさんの秘密鍵

解説

公開鍵暗号方式では，受信者の公開鍵で暗号化したデータは，対応する受信者の秘密鍵で復号する。したがってBさんの公開鍵で暗号化したメールはBさんの秘密鍵で復号する。
