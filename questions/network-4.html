QUESTION 031#center_style_fe_curated_085

10Mバイトのデータを100,000ビット/秒の回線で転送する。伝送効率が50%，1Mバイト＝10⁶バイトのとき，転送時間は何秒か。

  1. 0

    1,600

  2. 1

    200

  3. 2

    400

  4. 3

    800

正答と解説を確認

正答0. 1,600

解説

データ量は10×10⁶バイト＝80,000,000ビットである。実効速度は100,000×0.5＝50,000ビット/秒なので，転送時間は80,000,000÷50,000＝1,600秒である。

QUESTION 038#center_style_ip_curated_205

二つの拠点をWANで接続している。通信が使えなくなる可能性を下げ，可用性を高める対策として最も適当なものはどれか。

  1. 0

    別の通信事業者の回線を追加し，複数の回線で通信できるようにする。

  2. 1

    送信するパケットを暗号化してから送る。

  3. 2

    受信側でパケットの誤りを検出して訂正する。

  4. 3

    送信側でウイルス検査を行い，ウイルスを含まないパケットだけを送る。

正答0. 別の通信事業者の回線を追加し，複数の回線で通信できるようにする。

解説

別経路の回線を用意すると，一方の回線に障害が起きても通信を継続できる可能性が高まり，可用性が向上する。
