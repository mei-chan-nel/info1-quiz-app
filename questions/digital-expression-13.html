QUESTION 121#center_style_curated_081_001

10進数の7÷32を2進小数で表したものとして，最も適当なものはどれか。

  1. 0

    0.001011₂

  2. 1

    0.001101₂

  3. 2

    0.00111₂

  4. 3

    0.0111₂

正答と解説を確認

正答2. 0.00111₂

解説

7÷32は7/32である。2進小数では1/32は0.00001₂なので，7/32は4/32＋2/32＋1/32＝0.00100₂＋0.00010₂＋0.00001₂＝0.00111₂になる。

QUESTION 122#center_style_curated_081_002

アナログ信号をディジタル化してから処理する利点として，最も適当なものはどれか。

  1. 0

    ディジタル化するときに誤差がまったく発生しない。

  2. 1

    部品のばらつきや雑音の影響を受けにくく処理しやすい。

  3. 2

    数値を丸める処理をしても丸め誤差が発生しない。

  4. 3

QUESTION 123#center_style_curated_081_005

NANDは NOT(X AND Y) を表す。X OR Y と同じ結果を，NANDだけで表す式として最も適当なものはどれか。

  1. 0

    (X NAND Y) NAND X

  2. 1

    (X NAND X) NAND (Y NAND Y)

  3. 2

    (X NAND Y) NAND (X NAND Y)

  4. 3

    X NAND (Y NAND Y)

正答と解説を確認

正答1. (X NAND X) NAND (Y NAND Y)

解説

X NAND X は NOT(X AND X) なので NOT X になる。同様に Y NAND Y は NOT Y である。したがって (X NAND X) NAND (Y NAND Y) は NOT((NOT X) AND (NOT Y)) となり，ド・モルガンの法則から X OR Y と同じになる。

QUESTION 124#center_style_curated_082_001

16進数の小数0.248₁₆を10進数の分数で表したものとして，最も適当なものはどれか。

  1. 0

    31/512

  2. 1

    31/125

  3. 2

    31/32

  4. 3

    73/512

正答と解説を確認

正答3. 73/512

解説

16進小数の第1位は1/16，第2位は1/256，第3位は1/4,096を表す。0.248₁₆＝2/16＋4/256＋8/4,096＝64/512＋8/512＋1/512＝73/512である。

QUESTION 126#center_style_curated_083_001

任意の8ビットのデータXと 00001111 のXORをとる。結果として，最も適当なものはどれか。

  1. 0

    Xの上位4ビットはそのままで，下位4ビットの0と1が反転する。

  2. 1

    Xの上位4ビットの0と1が反転し，下位4ビットはそのままになる。

  3. 2

    Xの下位4ビットは，元の値に関係なくすべて1になる。

  4. 3

    Xの8ビットすべての0と1が反転する。

正答と解説を確認

正答0. Xの上位4ビットはそのままで，下位4ビットの0と1が反転する。

解説

XORは，相手のビットが0なら元の値を保ち，相手のビットが1なら0と1を反転する。00001111では上位4ビットが0，下位4ビットが1なので，上位4ビットはそのまま，下位4ビットだけが反転する。

QUESTION 130#center_style_curated_083_014

論理変数X，Yについて，X=0かつY=0のときだけ1になり，それ以外では0になる論理式として，最も適当なものはどれか。

  1. 0

    X AND Y

  2. 1

    NOT(X AND Y)

  3. 2

    NOT(X OR Y)

  4. 3

    X OR Y

正答と解説を確認

正答2. NOT(X OR Y)

解説

X OR Y は，XかYの少なくとも一方が1なら1になる。これをNOTで反転した NOT(X OR Y) は，X=0かつY=0のときだけ1になる。
