QUESTION 181#simple_original_092

次の組合せのうち、いずれもソフトウェアであるものはどれか。

  1. 0

    CPU・OS・Webブラウザ

  2. 1

    OS・Webブラウザ・表計算ソフト

  3. 2

    キーボード・マウス・表計算ソフト

  4. 3

    SSD・プリンタ・OS

正答と解説を確認

正答1. OS・Webブラウザ・表計算ソフト

解説

OS、Webブラウザ、表計算ソフトはいずれもプログラムであり、ソフトウェアに分類される。 OS、Webブラウザ、表計算ソフトはいずれもプログラムである。CPU、キーボード、SSD、プリンタは物理的な機器なので、同じ選択肢に含まれる組合せは除く。

QUESTION 182#simple_original_093

アプリケーションソフトがCPU、メモリ、ファイルなどを利用できるよう、資源を管理するソフトウェアはどれか。

  1. 0

    応用ソフトウェア

  2. 1

    ハードウェア

  3. 2

    基本ソフトウェア

  4. 3

    周辺装置

正答と解説を確認

正答2. 基本ソフトウェア

解説

基本ソフトウェアはOSなどを指し、ハードウェアの資源を管理してアプリケーションに共通の機能を提供する。 基本ソフトウェアはOSなどで、CPU・メモリ・ファイルといった資源を管理する。利用者の個別目的を支援する応用ソフトウェアとは役割が異なる。

QUESTION 184#simple_original_096

コンピュータの起動後に動作し、CPUの利用時間、メモリ、ファイル、入出力装置を管理するとともに、アプリケーションの実行環境を提供する基本ソフトウェアはどれか。

  1. 0

    OS

  2. 1

    BIOS

  3. 2

    デバイスドライバ

  4. 3

    ファームウェア

正答と解説を確認

正答0. OS

解説

OSはコンピュータ全体の資源を管理し、アプリケーションの実行環境を提供する。BIOS、デバイスドライバ、ファームウェアは、より限定された役割を担う。 BIOSは起動処理、デバイスドライバは特定機器の制御、ファームウェアは機器に組み込まれた制御を担う。コンピュータ全体の資源管理と実行環境の提供を担うのはOSである。

QUESTION 185#simple_original_100

電源を切った後もファイルを保存しておく用途に最も適した装置はどれか。

  1. 0

    CPU

  2. 1

    ディスプレイ

  3. 2

    SSD

  4. 3

    メインメモリ

正答と解説を確認

正答2. SSD

解説

SSDは電源を切ってもデータを保持する補助記憶装置である。メインメモリは処理中のデータを保持し、一般的には電源を切ると内容が失われる。 CPUは演算・制御、ディスプレイは出力を担う。メインメモリは高速に読み書きできるが一般に揮発性であり、長期保存にはSSDなどの不揮発性記憶装置を用いる。

QUESTION 186#simple_original_101

CPUが実行中のプログラムや処理途中のデータを一時的に置き、一般に電源を切ると内容が失われる記憶装置はどれか。

  1. 0

  3. 2

    出力装置

  4. 3

    主記憶装置

主記憶装置はCPUと直接やり取りして処理中の情報を保持する。一般的なRAMは揮発性である。 主記憶装置はCPUが実行中のプログラムやデータを直接扱う場所であり、一般的なRAMは揮発性である。SSDなどの補助記憶装置は長期保存に使う。

QUESTION 187#simple_original_102

プログラムで0.1を10回加算した結果が、表示上は1.0に近いが厳密には1.0と等しくならないことがある主な理由はどれか。

  1. 0

    0.1を2進浮動小数点で有限桁に正確には表せないため。

  2. 1

    小数の加算が禁止されているため。

  3. 2

    変数が文字列になるため。

  4. 3

    CPUが10進数を扱えないため。

正答と解説を確認

正答0. 0.1を2進浮動小数点で有限桁に正確には表せないため。

解説

多くの10進小数は2進数で有限桁にならず、浮動小数点では近似値として保持されるため丸め誤差が生じる。 0.1は2進数では有限桁で表せないため、浮動小数点では近似値として保持される。加算を重ねると丸め誤差が現れ、厳密な1.0との差になることがある。

QUESTION 188#simple_original_103

8ビット符号なし整数へ、表せる最大値255より大きい300を保存しようとしたときに起こる現象はどれか。

  1. 0

    アンダーフロー

  2. 1

    オーバーフロー

  3. 2

    デフラグ

  4. 3

    暗号化

正答と解説を確認

正答1. オーバーフロー

解説

オーバーフローは、数値型や記憶領域で表せる範囲を超えた値を扱おうとしたときに起こる。 8ビット符号なし整数で表せる範囲は0から255である。300はこの範囲を超えるため、表現可能な範囲を超えたときの現象であるオーバーフローが起こる。

QUESTION 189#simple2_original_021

デジタルデータの一般的な特徴として適切でないものを、次から選べ。

  1. 0

    同じデータを複製しても、原理上は内容を劣化させずにコピーできる。

  2. 1

    大量のデータを保存・検索・送信しやすい。

  3. 2

    複製したデータは、著作権や利用許諾にかかわらず自由に利用できる。

  4. 3

    用途に応じて編集や加工を行いやすい。

正答と解説を確認

正答2. 複製したデータは、著作権や利用許諾にかかわらず自由に利用できる。

解説

デジタルデータは、原理上劣化なく複製でき、保存・検索・送信・加工がしやすい。一方、この技術的な複製容易性と、法的・契約的に自由利用できるかどうかは別問題である。 データをコピーしても著作権は消えず、ライセンス条件も引き続き適用される。利用時には権利者の許諾、公開ライセンスの条件、引用などの権利制限規定を確認する。

QUESTION 190#simple2_original_022

101101₂+11011₂の計算結果を2進数で表しなさい。

  1. 0

    1001000₂

  2. 1

    1011000₂

  3. 2

    1001100₂

  4. 3

    1111000₂

正答と解説を確認

正答0. 1001000₂

解説

各数を10進数で確認すると、101101₂=32+8+4+1=45、11011₂=16+8+2+1=27である。したがって、和は45+27=72となる。 72=64+8=2⁶+2³なので、対応する桁を1にすると1001000₂である。
