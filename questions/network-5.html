QUESTION 041#center_style_curated_064_020

DNSの説明として，最も適当なものはどれか。

  1. 0

    インターネット上の情報を検索するための検索サービス

  2. 1

    ドメイン名やホスト名とIPアドレスを対応付ける仕組み

  3. 2

    オンラインショッピングで支払い金額を計算する仕組み

  4. 3

    個人のメールアドレスだけを管理する住所録

正答と解説を確認

正答1. ドメイン名やホスト名とIPアドレスを対応付ける仕組み

解説

DNSは，ドメイン名やホスト名とIPアドレスを対応付け，名前から通信先を見つけられるようにする仕組みである。

QUESTION 042#center_style_curated_064_023

PCとメールサーバの間で電子メールを送受信する。一般に，PCから送信するときと受信するときに使うプロトコルの組合せとして最も適当なものはどれか。

  1. 0

    送信にPOP，受信にSMTP

  2. 1

    送信にSMTP，受信にPOP

  3. 2

    送信にも受信にもPOP

  4. 3

    送信にも受信にもDNS

正答と解説を確認

正答1. 送信にSMTP，受信にPOP

解説

電子メールでは，一般に送信側からメールサーバへ送るときにSMTPを使い，受信側がメールサーバから受け取るときにPOPなどを使う。

QUESTION 043#center_style_curated_064_024

業務中に受け取った電子メールの添付ファイルを開いた直後，ソフトウェアが異常終了した。マルウェアを含む可能性があるとき，最初の対応として最も適当なものはどれか。

  1. 0

    PCをネットワークから切り離し，担当部門へ連絡する。

  2. 1

    すぐに同じ添付ファイルをもう一度開いて再現するか確かめる。

  3. 2

    原因を確かめる前に，関係者全員へ警告メールを一斉送信する。

  4. 3

    問題が起きたソフトウェアだけを再インストールして作業を続ける。

正答と解説を確認

正答0. PCをネットワークから切り離し，担当部門へ連絡する。

解説

マルウェア感染が疑われるときは，被害拡大を防ぐためにPCをネットワークから切り離し，決められた担当部門へ連絡するのが基本である。

QUESTION 044#center_style_curated_065_002

複数の顧客に同じ電子メールを送るとき，受信者同士に互いのメールアドレスを見せないようにしたい。宛先欄の使い方として最も適当なものはどれか。

  1. 0

    全員をTo欄に入れて送る。

  2. 1

    全員をCc欄に入れて送る。

  3. 2

    全員をBcc欄に入れて送る。

  4. 3

    本文の先頭に全員のメールアドレスを書いて送る。

正答と解説を確認

正答2. 全員をBcc欄に入れて送る。

解説

Bcc欄に入れた宛先は，他の受信者には表示されない。複数の相手に互いのメールアドレスを見せずに送るときに使う。

QUESTION 045#center_style_curated_065_004

    IPv4のネットワークとは必ず共存できないため
