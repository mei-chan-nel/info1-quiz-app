QUESTION 081#center_style_fe_curated_113

最上位ビットをパリティビットとする8ビットのデータから，下位7ビットだけを取り出したい。最も適当なビット演算はどれか。

  1. 0

    16進数7FとのANDをとる。

  2. 1

    16進数0FとのANDをとる。

  3. 2

    16進数7FとのORをとる。

  4. 3

    16進数80とのORをとる。

正答と解説を確認

正答0. 16進数7FとのANDをとる。

解説

16進数7Fは2進数で01111111なので，ANDをとると最上位ビットだけが0になり，下位7ビットはそのまま残る。

QUESTION 082#center_style_ap_curated_127

二つの真偽値A，Bについて，AとBが異なるときだけ真になる演算はXORである。XORの結果を反転した演算の説明として，最も適当なものはどれか。

  1. 0

    AとBが同じときに真になる等価演算

  2. 1

    AとBの少なくとも一方が真のときに真になるOR

  3. 2

    AとBがともに真のときだけ真になるAND

  4. 3

    AとBの少なくとも一方が真のときに偽になる否定論理和

正答と解説を確認

正答0. AとBが同じときに真になる等価演算

解説

XORはAとBが異なるときに真，同じときに偽になる。その結果を反転すると，AとBが同じときに真になる等価演算になる。

QUESTION 083#center_style_ip_curated_223

  3. 2

    TB < GB < MB < KB

QUESTION 084#center_style_ip_curated_224

AからZまでの26種類の文字を，すべて異なるビット列で表したい。最小で何ビット必要か。

  1. 0

    4ビット

  2. 1

    5ビット

  3. 2

    6ビット

  4. 3

    7ビット

正答と解説を確認

正答1. 5ビット

解説

nビットでは2ⁿ種類を表せる。4ビットでは2⁴＝16種類で26種類に足りないが、5ビットでは2⁵＝32種類を表せる。したがって、最小で5ビット必要である。

QUESTION 087#center_style_ip_curated_227

画面の解像度を1280×960ピクセルから1024×768ピクセルに下げ，同じ大きさの画面全体に表示した。このとき起こりやすい変化として最も適当なものはどれか。

  1. 0

    画面上の文字や画像が大きく見えやすくなる。

  2. 1

    画面に表示できる情報量が増える。

  3. 2

QUESTION 088#center_style_ap_curated_136

D-A変換器で、入力値0のとき出力電圧は0.5Vであり、入力値が1増えるごとに0.02V増える。2進数1010を入力したときの出力電圧は何Vか。

  1. 0

    0.2V

  2. 1

    0.5V

  3. 2

    0.7V

  4. 3

    1.0V

正答と解説を確認

正答2. 0.7V

解説

2進数1010は10進数で10である。入力値0のとき0.5Vで、1段階当たり0.02V増えるので、出力電圧は0.5＋10×0.02＝0.7Vとなる。

QUESTION 089#center_style_ip_curated_241

「札幌にある日本料理の店」と「函館にある日本料理の店」をまとめて検索したい。検索条件として最も適当なものはどれか。

  1. 0

    札幌 AND 函館 AND 日本料理

  2. 1

    (札幌 OR 函館) AND 日本料理

  3. 2

    札幌 AND (函館 OR 日本料理)

  4. 3

    札幌 OR 函館 OR 日本料理

正答と解説を確認

正答1. (札幌 OR 函館) AND 日本料理

解説

場所は札幌または函館のどちらかでよいのでORで結び，料理の種類は日本料理に限定するのでANDで結ぶ。

QUESTION 090#center_style_curated_063_011

8ビットの値で，00001000に対応するビットだけを1にし，他のビットは元のままにしたい。元の値に対して行う演算として最も適当なものはどれか。

  1. 0

    00001000とのAND演算

  2. 1

    00001000とのOR演算

  3. 2

    01001000とのAND演算

  4. 3

    01001000とのOR演算

正答と解説を確認

正答1. 00001000とのOR演算

解説

OR演算では，1とORをとったビットは必ず1になり，0とORをとったビットは元の値のまま残る。したがって00001000とのOR演算が適切である。
