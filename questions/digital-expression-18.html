QUESTION 171#simple_original_069

1280×720画素、24ビットカラー、毎秒30フレームの無圧縮動画を60秒記録する。データ量は何バイトか。

  1. 0

    82,944,000バイト

  2. 1

    2,764,800,000バイト

  3. 2

    49,766,400,000バイト

  4. 3

    4,976,640,000バイト

正答と解説を確認

正答3. 4,976,640,000バイト

解説

1フレームは1280×720×3=2,764,800バイトである。これを30フレーム×60秒分記録する。 1フレームは1280×720×3=2,764,800バイトである。これを毎秒30フレーム、60秒分記録するため、さらに30×60を掛ける。

  2. 1

  3. 2

    105,840,000バイト

  4. 3

    423,360,000バイト

正答と解説を確認

正答0. 52,920,000バイト

44,100×16÷8×2チャンネル×300秒=52,920,000バイトである。 1秒当たりのデータ量は44,100×16÷8×2チャンネルで求める。5分は300秒なので、この値に300を掛けて52,920,000バイトとなる。

QUESTION 175#simple_original_073

写真を小さく保存するため、細部の情報を一部捨てて圧縮した。この方式の特徴として最も適切なものはどれか。

  1. 0

    展開後に元のビット列を完全に復元できる。

  2. 1

    文書の保存に用いる圧縮方式である。

  3. 2

    圧縮前後でデータ量が同じである。

  4. 3

    展開しても元のデータを完全には復元できない。

正答と解説を確認

正答3. 展開しても元のデータを完全には復元できない。

解説

非可逆圧縮は、人の知覚に影響の小さい情報などを捨ててデータ量を減らすため、完全復元はできない。 非可逆圧縮は一部の情報を捨てて容量を小さくするため、展開しても元のビット列には戻らない。完全に復元できるのは可逆圧縮の特徴である。
