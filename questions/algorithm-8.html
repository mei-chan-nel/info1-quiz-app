QUESTION 071#center_style_curated_083_013

プログラムの実行方式について，ソースプログラムを1行ずつ解釈しながら実行する方式と，実行用のプログラムに翻訳してから実行する方式の組合せとして，最も適当なものはどれか。

  1. 0

    インタプリタ方式，コンパイラ方式

  2. 1

    コンパイラ方式，インタプリタ方式

  3. 2

    インタプリタ方式，インタプリタ方式

  4. 3

    コンパイラ方式，コンパイラ方式

正答と解説を確認

正答0. インタプリタ方式，コンパイラ方式

解説

インタプリタ方式はソースプログラムを解釈しながら実行する方式である。コンパイラ方式は，あらかじめ実行用のプログラムに翻訳してから実行する方式である。

QUESTION 072#center_style_curated_084_006

気温表から，年が2024で，月が7月または8月で，最高気温が35度以上の行を抽出したい。条件式として，最も適当なものはどれか。

  1. 0

    年=2024 AND 月=7 AND 月=8 AND 最高気温>=35

  2. 1

    年=2024 AND (月=7 OR 月=8) AND 最高気温>=35

  3. 2

    年=2024 OR (月=7 AND 月=8) OR 最高気温>=35

  4. 3

    年=2024 AND (月=7 OR 月=8 OR 最高気温>=35)

正答と解説を確認

正答1. 年=2024 AND (月=7 OR 月=8) AND 最高気温>=35

解説

同じ行の月が7月であり同時に8月であることはないので，月の条件はORで結ぶ。年と月の範囲と最高気温の条件はすべて満たす必要があるのでANDで結ぶ。

QUESTION 074#center_style_curated_085_016

ペンを下ろし，次の処理を繰り返して1辺1mの正六角形を描くプログラムを考える。 「1m直進する。右にA度向きを変える。これをB回行う。」 A，Bの組合せとして，最も適当なものはどれか。

  1. 0

    A=60，B=6

  2. 1

    A=90，B=4

  3. 2

    A=120，B=3

  4. 3

    A=45，B=8

正答と解説を確認

正答0. A=60，B=6

解説

正六角形は6本の辺から成り，外角は360度÷6=60度である。したがって，1m直進して右に60度曲がる処理を6回繰り返せばよい。

QUESTION 075#simple_original_097

Python、JavaScript、Cの共通点として最も適切なものはどれか。

  1. 0

    画像ファイル形式。

  2. 1

    コンピュータへ処理手順を記述するプログラミング言語。

  3. 2

QUESTION 076#simple_original_113

プログラムの基本的な実行の流れを構成する三つの制御構造はどれか。

  1. 0

    入力・出力・記憶

  2. 1

    加算・減算・乗算

  3. 2

    順次・分岐・反復

  4. 3

    検索・整列・圧縮

正答と解説を確認

正答2. 順次・分岐・反復

解説

構造化プログラミングでは、処理を順に行う順次、条件で選ぶ分岐、繰り返す反復が基本となる。 構造化プログラミングの基本は、順に実行する順次、条件で選ぶ分岐、繰り返す反復である。入出力や演算は処理内容であり、制御の構造そのものではない。

  2. 1

    コンパイラ

  3. 2

    デバッグ

QUESTION 079#simple_original_132

数理モデルの例として最も適切なものはどれか。

  1. 0

    建物の形状を縮尺模型で表す。

  2. 1

    人口の増減を、出生率・死亡率・移動率を含む数式で表す。

  3. 2

    駅と乗換えの関係を路線図で表す。

  4. 3

    製品の外観を実物大の模型で表す。

正答1. 人口の増減を、出生率・死亡率・移動率を含む数式で表す。

QUESTION 080#simple2_original_053

次の文章の（ア）・（イ）に当てはまる語句の組合せとして、最も適切なものを選べ。 プログラミング言語を用いて、人が読んだり修正したりできる形で記述した命令の集まりを（ア）という。（ア）は処理系によって変換または解釈され、その処理の最終段階では、CPUが直接実行できる（イ）の命令が実行される。

  1. 0

    （ア）アルゴリズム　　（イ）アセンブリ言語

  2. 1

    （ア）ソースコード　　（イ）機械語

  3. 2

    （ア）プログラム　　　（イ）プログラミング言語

  4. 3

    （ア）命令コード　　　（イ）高級言語

正答と解説を確認

正答1. （ア）ソースコード　　（イ）機械語

解説

（ア）はソースコード、（イ）は機械語である。ソースコードは、PythonやCなどのプログラミング言語を用いて、人が読んで記述・修正しやすい形で命令を表したものをいう。 CPUが直接解釈して実行するのは、そのCPUの命令体系に対応するビット列で表された機械語である。コンパイラ方式ではソースコードをあらかじめ機械語などへ変換する。インタプリタ方式では処理系がソースコードを逐次解釈し、その処理を実現する機械語の命令をCPUに実行させる。
