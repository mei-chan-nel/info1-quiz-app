QUESTION 271#original_knowledge_0001

情報セキュリティ上のソーシャルエンジニアリングに当たる行為はどれか。

  1. 0

    社員を装い、電話でパスワード再設定用の情報を聞き出す。

  2. 1

    拾った他人のIDとパスワードを使ってSNSへログインする。

  3. 2

    大量の通信を送り付けてサーバの応答を妨げる。

  4. 3

    ファイルを暗号化して復旧と引換えに金銭を要求する。

正答と解説を確認

正答0. 社員を装い、電話でパスワード再設定用の情報を聞き出す。

解説

人の心理や行動の隙を利用し、会話などの非技術的な手段で秘密情報を得ている。 ソーシャルエンジニアリングは、システムの技術的な弱点ではなく、人の思い込み、親切心、権威への弱さなどを利用して秘密情報を得る手口である。電話で社員を装って認証情報を聞き出す、画面やキー入力を肩越しに盗み見る、廃棄書類から情報を探すといった行為が例となる。拾った認証情報で実際にログインする行為は不正アクセス、ファイルを暗号化して金銭を要求する行為はランサムウェアであり、区別が必要である。

QUESTION 272#original_knowledge_0002

他人のSNSのIDとパスワードが書かれたメモを拾った。本人の許可なくログインする行為について、最も適切な説明はどれか。

  1. 0

    データを変更しなければ不正アクセスではない。

  2. 1

    無断でのログインは、不正アクセスに当たり得る。

  3. 2

    一度だけのログインなら不正アクセスではない。

  4. 3

    自分で盗んだパスワードでなければ使える。

正答と解説を確認

正答1. 無断でのログインは、不正アクセスに当たり得る。

解説

正当な利用権限がないのに他人のID・パスワードでアクセスする行為は、確認目的でも正当化されない。 他人のユーザID・パスワードを無断で使用する行為（なりすまし）は不正アクセス禁止法により禁止されている。 また、不正アクセス禁止法は、他人のユーザID・パスワードを無断で第三者に提供する行為（助長する行為）、システムの不備（セキュリティホール）を利用して、コンピュータを利用する行為（攻撃する行為）も禁じている。

QUESTION 273#original_knowledge_0003

パブリシティ権として最も適切な説明はどれか。

  1. 0

    著作者の氏名表示を決める著作者人格権。

  2. 1

    写真に写った建物や物の所有権を保護する権利。

  3. 2

    著名人の氏名や肖像が持つ商業的価値を保護する権利。

  4. 3

    写真データの画質や解像度を保護する権利。

正答と解説を確認

正答2. 著名人の氏名や肖像が持つ商業的価値を保護する権利。

解説

著名人の氏名・肖像などが持つ経済的価値を無断で利用する場面では、パブリシティ権が問題となる。 肖像に関する人格的利益は、一般の人も含め、みだりに撮影・公表されない利益を保護する。撮影目的を説明して本人の同意を得た集合写真は、無断撮影とは事情が異なる。一方、パブリシティ権は、著名人の氏名や肖像が持つ顧客吸引力・商業的価値に着目した権利である。アイドルの写真を無断で集め、商品として販売する行為では、その商業的価値の利用が問題になる。

QUESTION 274#original_knowledge_0004

意匠権の保護対象として最も適切なものはどれか。

  1. 0

    自然法則を利用した技術的な発明。

  2. 1

    商品やサービスを識別する文字やロゴ。

  3. 2

    小説や音楽などの創作的な表現。

  4. 3

    物品・建築物・画像などに現れるデザイン。

正答と解説を確認

正答3. 物品・建築物・画像などに現れるデザイン。

解説

特許は発明、意匠は物品等のデザイン、商標は商品・サービスの識別標識を主に保護する。 産業財産権は、技術的な創作、デザイン、営業上の標識などを保護する制度である。特許権は自然法則を利用した技術的な発明、実用新案権は物品の形状・構造・組合せに関する考案を対象とする。意匠権は物品・建築物・画像などの視覚を通じたデザインを、商標権は商品やサービスを他人のものと区別するための文字・図形・ロゴなどを保護する。

QUESTION 275#original_knowledge_0005

著作権と産業財産権の権利発生について、最も適切なものはどれか。

  1. 0

    著作権は創作時に、特許権は原則として登録により発生する。

  2. 1

    著作権も特許権も、特許庁への届出で発生する。

  3. 2

    著作権も特許権も、公表した時点で自動発生する。

  4. 3

    著作権は販売時、特許権は発明者の死亡時に発生する。

正答と解説を確認

正答0. 著作権は創作時に、特許権は原則として登録により発生する。

解説

著作権は無方式主義であり、産業財産権は原則として所定の手続によって権利を得る。 著作権は著作者が著作物を創作した時点で自動的に発生する権利である。 権利を取得するために何らかの方式や手続きは必要ない。 これを無方式主義という。 一方、産業財産権（特許権、実用新案権、意匠権、商標権）は特許庁に出願し認められた時点から得ることができる。

QUESTION 276#original_knowledge_0006

著作権で保護される対象として最も適切なものはどれか。

  1. 0

    物語の抽象的な設定やアイデアそのもの。

  2. 1

    具体的に表現された、創作性のある文章や絵。

  3. 2

    自然法則や歴史的事実そのもの。

  4. 3

    他人の作品をそのまま複製する行為。

正答と解説を確認

正答1. 具体的に表現された、創作性のある文章や絵。

解説

著作権は具体的な創作的表現を保護し、抽象的なアイデアや単なる事実そのものを独占させる制度ではない。 著作物とは「①思想又は感情を ②創作的に ③表現した ④文芸、学術、美術又は音楽の範囲に属するもの」である。 作品の背景である世界観は「アイデア」に分類され、③表現したものとは見做されず著作物に該当しない。 ただし、世界観や設定を超えた模倣を行った場合、著作権侵害となる可能性がある。 数式や単なるデータ、歴史的事実は「思想・感情」と関わりがないので、著作物から除かれる。 素材の選択又は配列によって創作性を有するものは編集著作物として保護される。 新聞や雑誌は編集著作物の代表例である。 著作物となるジャンル分けは厳密なものではなく、知的・文化的に含まれる広い範囲で認められる。 コンピュータプログラムも著作物である。 地図を「文芸、学術、美術又は音楽」のいずれかに分類することは難しいが、著作物として認められている。

QUESTION 277#original_knowledge_0007

市販キャラクターを無断で複製したクラスTシャツを生徒全員で使う行為について、最も適切な説明はどれか。

  1. 0

    学校内で使うなら、常に自由に複製できる。

  2. 1

    利益を得なければ、どのような無断利用もできる。

  3. 2

    非営利でも、当然に私的使用の範囲になるとはいえない。

  4. 3

    クラス全体で使っても、必ず私的使用に当たる。

正答と解説を確認

正答2. 非営利でも、当然に私的使用の範囲になるとはいえない。

解説

私的使用のための複製は個人的・家庭内などの限られた範囲であり、クラス全体での利用は当然には含まれない。 私的使用のための複製は、個人・家庭内などの限られた範囲で利用する場合の例外である。クラスTシャツを作って文化祭などで全員が使う行為は、その範囲を超えるため、私的使用の例外が当然に当てはまるわけではない。営利目的でないことだけで、他人の著作物を自由に複製できるわけではなく、学校教育での利用に関する別の要件とも区別して考える必要がある。

QUESTION 278#original_knowledge_0008

既存曲を演奏する際、著作権者の許諾が不要となり得る条件として最も適切なものはどれか。

  1. 0

    非営利で、聴衆無料だが、実演者へ報酬を払う。

  2. 1

    営利目的で、聴衆から料金を取り、報酬も払わない。

  3. 2

    非営利で、聴衆から料金を取り、報酬を払わない。

  4. 3

    非営利で、聴衆から料金を取らず、実演者に報酬を払わない。

正答と解説を確認

正答3. 非営利で、聴衆から料金を取らず、実演者に報酬を払わない。

解説

非営利・無料・無報酬の上演等には例外がある一方、公衆送信は別の利用行為として判断する必要がある。 著作権法上、営利を目的とせず、聴衆から料金を取らず、実演者などに報酬を支払わない上演・演奏・上映・口述は、一定の条件の下で許諾なしに行える。ただし、この例外は利用方法を限定している。演奏を録画してインターネットで配信する行為は公衆送信に当たり、同じ例外で当然に許されるわけではなく、原則として別途の権利処理を検討する必要がある。

QUESTION 279#original_knowledge_0009

レポートで他人の文章を引用として利用する場合の説明として、最も適切なものはどれか。

  1. 0

    自分の文章を主とし、引用部を区別して出所を示す。

  2. 1

    引用部を主とし、出所を示さずに転載する。

  3. 2

    本文と区別せず、出所も示さずに転載する。

  4. 3

    未公表の原稿を、本人に無断で自由に転載する。

正答と解説を確認

正答0. 自分の文章を主とし、引用部を区別して出所を示す。

解説

引用には、公表済みであること、引用の必然性、明確な区別、主従関係、出所明示などが重要である。 引用は、公表された著作物を、公正な慣行に合致し、報道・批評・研究などの目的上正当な範囲で利用するための例外である。自分の文章が主で引用部分が従となること、引用部分をかぎ括弧や字下げなどで明確に区別すること、出所を示すことが重要になる。単に出典を書いたり、改変しなかったりするだけで、長文の無断転載が引用になるわけではない。

QUESTION 280#original_knowledge_0010

小説を原作とするアニメを第三者が利用したい。権利処理について、最も適切なものはどれか。

  1. 0

    アニメ化された時点で、小説の著作権は自動的に消滅する。

  2. 1

    利用態様により、原作とアニメの両方の権利が関係する。

  3. 2

    原作者の許諾だけあれば、アニメ制作側の権利は常に無視できる。

  4. 3

    二次的著作物には著作権が一切認められない。

正答と解説を確認

正答1. 利用態様により、原作とアニメの両方の権利が関係する。

解説

二次的著作物の利用では、二次的著作物の著作者に加えて、原著作物の著作者の権利も関係し得る。 二次的著作物とは、著作物を翻案（翻訳、編曲、変形、映画化・アニメ化）することで創作した著作物のことである。 原作著作物の著作者は、翻案（二次的著作物の創作すること）だけでなく、二次的著作物を第三者が利用することに関しても、それを許諾するかしないかを決めることができる（二次的著作物の利用権）。
