QUESTION 062#center_style_ap_curated_108

1ビットごとの誤り率が10%の通信で，同じビットを3回送り，多数決で元のビットを復元する。復元後のビットが誤る確率として，最も近いものはどれか。

  1. 0

    2.8%

  2. 1

    1.0%

  3. 2

    3.1%

  4. 3

    10.0%

正答と解説を確認

正答0. 2.8%

解説

多数決が誤るのは，3回のうち2回または3回が誤る場合である。確率は3×0.1²×0.9+0.1³＝0.027+0.001＝0.028，つまり2.8%である。

QUESTION 064#center_style_ap_curated_110

  2. 1

    820

  3. 2

    1,024

  4. 3

    1,312

QUESTION 065#center_style_ap_curated_112

15Mバイトのプログラムを圧縮率40%で保存している。圧縮データを20Mバイト/秒で読み込み，1Mバイトの圧縮データの展開に0.03秒かかる。読み込みと展開を同時に行わないとき，全体で何秒かかるか。

  1. 0

    0.48

  2. 1

    0.75

  3. 2

    0.93

  4. 3

    1.20

正答と解説を確認

正答0. 0.48

解説

圧縮後の大きさは15×0.40=6Mバイトである。読み込み時間は6/20=0.30秒，展開時間は6×0.03=0.18秒なので，合計は0.48秒である。

QUESTION 066#center_style_fe_curated_084

1画素を24ビットで表すカラー画像で表現できる色の数は、1画素を8ビットで表す画像の何倍か。

  1. 0

    256倍

  2. 1

    65,536倍

  3. 2

    16倍

  4. 3

    16,777,216倍

正答と解説を確認

正答1. 65,536倍

解説

nビットで表現できるパターン数は2ⁿ通りである。ビット数の差は24－8＝16なので、色数の比は2²⁴÷2⁸＝2¹⁶＝65,536倍となる。

QUESTION 067#center_style_fe_curated_086

1Mバイトのメモリをバイト単位で区別してアクセスするには，最低何本のアドレス用の信号線が必要か。ここで，1Mバイト=1,024×1,024バイトとする。

  1. 0

    20

  2. 1

    18

  3. 2

    19

  4. 3

    21

正答と解説を確認

正答0. 20

解説

1Mバイトは1,024×1,024＝2¹⁰×2¹⁰＝2²⁰バイトである。2²⁰個の場所を区別するには20ビット分のアドレスが必要なので、信号線は20本である。
