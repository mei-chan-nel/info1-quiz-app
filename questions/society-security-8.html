QUESTION 071#center_style_ip_curated_103

学校行事で撮影した生徒の顔がはっきり分かる写真を，学校のWebページに掲載したい。特に確認すべき権利として最も適当なものはどれか。

  1. 0

    肖像権

  2. 1

    商標権

  3. 2

    特許権

  4. 3

    実用新案権

正答と解説を確認

正答0. 肖像権

解説

肖像権は，無断で写真を撮られたり，公表・利用されたりしない権利に関係する。顔が分かる写真を公開する場合は本人や保護者の同意などを確認する必要がある。

QUESTION 072#center_style_ap_curated_082

CCライセンスの説明として最も適当なものはどれか。

  1. 0

    著作者が，作品の再利用を許す条件をあらかじめ分かりやすく示すための仕組みである。

  2. 1

    著作物は登録しなければ保護されないという制度である。

  3. 2

    著作物を利用するとき，出所を示せばどのような使い方でも必ず許される制度である。

  4. 3

    著作権の保護期間をなくし，すべての著作物を自由に使えるようにする制度である。

正答と解説を確認

正答0. 著作者が，作品の再利用を許す条件をあらかじめ分かりやすく示すための仕組みである。

解説

CCライセンスは，著作者が再利用の条件をあらかじめ示し，利用者が条件を理解しやすくするための仕組みである。

QUESTION 074#center_style_ip_curated_114

組織がソーシャルメディアポリシーを定める目的として最も適当なものはどれか。

  1. 0

    組織や従業員がソーシャルメディアを使う際の方針やルールを示し，トラブルを防ぎやすくするため

  2. 1

    従業員が投稿した内容を，内容にかかわらず組織がすべて自動的に削除するため

  3. 2

    私的な投稿を含め，何を書いても組織や本人が責任を負わないことにするため

  4. 3

    投稿の表示回数やフォロワー数だけを増やすため

正答と解説を確認

正答0. 組織や従業員がソーシャルメディアを使う際の方針やルールを示し，トラブルを防ぎやすくするため

解説

ソーシャルメディアポリシーは，組織や従業員がソーシャルメディアを利用するときの考え方やルールを示し，不適切な発信や誤解によるトラブルを防ぎやすくするために定める。

QUESTION 075#center_style_ap_curated_095

日本で産業財産権と総称される権利の組合せとして，最も適当なものはどれか。

    意匠権，実用新案権，商標権，特許権

  2. 1

    意匠権，実用新案権，著作権，特許権

  3. 2

QUESTION 079#center_style_ip_curated_136

情報システムを安定して動かすための事前対策を考えるときの考え方として，最も適当なものはどれか。

  1. 0

    予想される損失と対策コストのトレードオフを考えて，必要な対策を選ぶ。

  2. 1

    バックアップを取っていれば，他の対策は不要である。

  3. 2

    自然災害は予測できないので，事前対策を考える必要はない。

  4. 3

    外部からの不正アクセスだけを防げば，安定稼働の対策として十分である。

正答と解説を確認

正答0. 予想される損失と対策コストのトレードオフを考えて，必要な対策を選ぶ。

解説

対策には費用や手間がかかるため，予想される損失，発生可能性，対策コストのトレードオフを考えて選ぶ必要がある。
