QUESTION 091#center_style_curated_083_002

URLの説明として，最も適当なものはどれか。

  1. 0

    Web上のページやファイルなどの場所を示すための表記である。

  2. 1

    Webページの本文や見出しをタグで記述するための言語である。

  3. 2

    ブラウザとWebサーバが通信するときに使う規則そのものである。

  4. 3

    Webページが更新された履歴を利用者へ知らせるメッセージである。

正答と解説を確認

正答0. Web上のページやファイルなどの場所を示すための表記である。

解説

URLは，Web上のページやファイルなどの情報資源の場所を示す表記である。HTMLのような記述言語や，HTTPのような通信手順そのものとは区別する。

QUESTION 098#original_knowledge_0040

LANとして最も適切な説明はどれか。

  1. 0

    地理的に離れたLAN同士を結ぶ広域ネットワーク。

  2. 1

    Web上の名前をIPアドレスに対応付ける仕組み。

  3. 2

    Web通信を暗号化するためのプロトコル。

  4. 3

    学校や家庭など、比較的限られた範囲を結ぶネットワーク。

正答と解説を確認

正答3. 学校や家庭など、比較的限られた範囲を結ぶネットワーク。

解説

LANはLocal Area Networkの略で、限られた場所にある機器を接続するネットワークである。 LAN(Local Area Network)は家庭内や建物などの一定の狭い範囲内における、コンピュータやサーバ、プリンター、ゲーム機などの機器が接続されたネットワークである。 LANの構築・管理はユーザ自らが行う。 離れた場所にある複数のLAN同士を接続するネットワークをWAN(Wide Area Network：広域通信網)という。 さらに、不特定多数のネットワーク同士を繋いだ巨大なネットワークがインターネットである。 インターネットへの接続するにはISP（Internet Service Provider）と契約する必要がある。

QUESTION 099#original_knowledge_0041

WANとして最も適切な説明はどれか。

  1. 0

    地理的に離れたLANや拠点を結ぶ広域ネットワーク。

  2. 1

    一つの家庭や学校内だけを結ぶネットワーク。

  3. 2

    端末の画面表示を担当するソフトウェア。

  4. 3

    ファイルを圧縮して容量を小さくする方式。

正答と解説を確認

正答0. 地理的に離れたLANや拠点を結ぶ広域ネットワーク。

解説

WAN（Wide Area Network）は、地理的に離れたLANや拠点を結ぶ広域ネットワークである。例えば、本社と支社のLANを通信事業者の回線やVPNなどで接続する構成が当てはまる。家庭や一つの建物など、比較的限られた範囲を結ぶLANとは、対象とする地理的範囲が異なる。

QUESTION 100#original_knowledge_0042

Ethernetとして最も適切な説明はどれか。

  1. 0

    IEEE 802.11系の無線LAN機器の認証ブランド。

  2. 1

    主に有線LANで用いられる代表的な通信規格。

  3. 2

    ホスト名からIPアドレスを調べるサービス。

  4. 3

    電子メールを配送するためのサーバ。

正答と解説を確認

正答1. 主に有線LANで用いられる代表的な通信規格。

解説

Ethernetは、主に有線LANで広く用いられる通信規格群であり、IEEE 802.3として標準化されている。LAN内ではMACアドレスを用いてフレームを届ける。IEEE 802.11系の無線LAN規格や、その相互接続性に関するWi‑Fiブランドとは区別する。
