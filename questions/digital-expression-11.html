QUESTION 101#center_style_curated_071_009

3ビットのデータ101に，1の個数が偶数になるようにパリティビットを1ビット付ける。付けるビットとして最も適当なものはどれか。

  1. 0

    0

  2. 1

    1

  3. 2

    101

  4. 3

    どちらでもよい

正答と解説を確認

正答0. 0

解説

101には1が2個あり，すでに偶数である。偶数パリティでは，付けた後も1の個数を偶数にするので，付けるビットは0である。

QUESTION 102#center_style_curated_072_001

音声を1秒間に11,000回サンプリングし，1回の値を8ビットで記録する。512×10⁶バイトの記憶媒体に記録できる音声は最大でおよそ何分か。

  1. 0

    77分

  2. 1

    96分

  3. 2

    776分

  4. 3

    969分

正答と解説を確認

正答2. 776分

解説

1秒のデータ量は11,000×8＝88,000ビット＝11,000バイトである。記録できる時間は，512×10⁶÷11,000÷60≒775.8分なので，およそ776分である。

QUESTION 103#center_style_curated_072_002

64,000ビット/秒の回線で，平均1000バイトのファイルを2秒ごとに送る。制御情報として20%分が追加されるとき，回線利用率は何%か。

  1. 0

    0.9%

  2. 1

    6.3%

  3. 2

    7.5%

  4. 3

    30.0%

正答と解説を確認

正答2. 7.5%

解説

1回に送る量は1000×1.2=1200バイト，すなわち9600ビットである。2秒ごとなので1秒当たり4800ビットを使い，4800÷64000=0.075，つまり7.5%である。

QUESTION 105#center_style_curated_072_006

2,000バイトのデータを2件ずつまとめ，400バイトの見出し情報を付けて送る。データは1時間に100,000件発生し，回線速度は1Mビット/秒である。1Mビット＝10⁶ビットとして，回線利用率はおよそ何%か。

  1. 0

    6.1%

  2. 1

    44%

  3. 2

    49%

  4. 3

    53%

正答と解説を確認

解説

2件ごとに送る量は2,000×2＋400＝4,400バイトである。1時間に50,000回送るので，1時間の送信量は4,400×8×50,000ビットである。1秒当たりでは約488,889ビットとなり，10⁶ビット/秒に対する利用率は約48.9%なので，およそ49%である。

QUESTION 108#center_style_curated_072_015

アナログ音声をディジタル化するとき，元の波形により近く復元しやすい組合せとして最も適当なものはどれか。

  1. 0

    サンプリング周期を長くし，量子化の段階数を多くする

  2. 1

    サンプリング周期を長くし，量子化の段階数を少なくする

  3. 2

    サンプリング周期を短くし，量子化の段階数を多くする

  4. 3

    サンプリング周期を短くし，量子化の段階数を少なくする

正答と解説を確認

正答2. サンプリング周期を短くし，量子化の段階数を多くする

解説

サンプリング周期が短いほど細かい時間間隔で値を読み取れ，量子化の段階数が多いほど振幅を細かく表せる。そのため，元の波形に近く復元しやすいのは，周期を短くし，段階数を多くする組合せである。
