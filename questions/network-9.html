QUESTION 081#center_style_curated_073_005

パケット交換方式について，適当なものだけを選んだ組合せはどれか。a: インターネットの通信で使われる。b: 通信経路をずっと占有する。c: 通信量は実際に送受信したパケットの大きさなどを基に考えられる。

  1. 0

    a，b

  2. 1

    a，c

  3. 2

    b，c

  4. 3

    a，b，c

正答と解説を確認

正答1. a，c

解説

パケット交換方式は，データを小さなまとまりに分けて送る方式で，インターネットの通信で使われる。通信経路をずっと占有する方式ではないので，aとcが適当である。

QUESTION 084#center_style_curated_073_014

QUESTION 086#center_style_curated_074_004

チェーンメールの特徴として適当なものだけを選んだ組合せはどれか。a: グループ内の連絡や情報共有を目的とする。b: ネットワークやサーバに無駄な負荷をかける。c: 返信に対する返信を何度も繰り返す。d: 本文中に多数への転送をあおる文言がある。

  1. 0

    a，b

  2. 1

    a，d

  3. 2

    cだけ

  4. 3

    b，d

正答と解説を確認

正答3. b，d

解説

チェーンメールは，多数への転送を求める内容によって広がり，ネットワークやサーバに余計な負荷をかけることがある。したがってbとdを含むものが正答である。

QUESTION 088#center_style_curated_077_001

IoTの活用例として，適当でないものはどれか。

  1. 0

    工場の機械にセンサと通信機能を付け，稼働状況を離れた場所で確認する。

  2. 1

    自動車の位置情報を集め，渋滞しやすい場所の情報を配信する。

  3. 2

    電力メータが使用量を自動で電力会社へ送信する。

  4. 3

    ネットワークに接続せず，単独のコンピュータで文書を作成する。

正答と解説を確認

正答3. ネットワークに接続せず，単独のコンピュータで文書を作成する。

解説

IoTは，さまざまなモノをインターネットなどのネットワークへ接続し，状態の収集や遠隔制御に役立てる考え方である。工場設備の監視，位置情報を用いた交通情報の配信，電力使用量の自動送信はいずれもIoTの例である。ネットワークに接続しない単独のコンピュータで文書を作成するだけの例は，IoTには当たらない。

QUESTION 089#center_style_curated_082_002

IoTで扱う「モノ」の説明として，最も適当なものはどれか。

  1. 0

    ネットワークにつながり，データの送受信や制御に関わる機器や対象物である。

Webページの構造・見た目・動作を役割ごとに分けて作成する。HTMLが主に担当する内容として，最も適当なものはどれか。

  1. 0

    見出し，段落，リンクなど，ページの内容と文書構造を記述する。

  2. 1

    文字色，余白，配置など，ページの見た目をまとめて指定する。
