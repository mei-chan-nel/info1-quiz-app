QUESTION 131#center_style_curated_083_019

論理変数A，Bについて，A XOR B と同じ結果になる式として，最も適当なものはどれか。

  1. 0

    (A OR B) AND (A OR (NOT B))

  2. 1

    (A OR B) AND ((NOT A) OR (NOT B))

  3. 2

    ((NOT A) OR B) AND (A OR (NOT B))

  4. 3

    ((NOT A) OR B) AND ((NOT A) OR (NOT B))

正答と解説を確認

正答1. (A OR B) AND ((NOT A) OR (NOT B))

解説

A XOR B は，AとBのどちらか一方だけが1のとき1になる。正答の内容は，少なくとも一方が1であり，かつ両方が1ではないことを表すのでXORと同じである。

QUESTION 132#center_style_curated_084_007

3ビットの属性情報で，左から順に読取り，書込み，削除の許可を表し，1は許可，0は不許可を表す。属性情報が101のとき，許可される操作として最も適当なものはどれか。

  1. 0

    読取りと削除

  2. 1

    読取りと書込み

  3. 2

    書込みと削除

  4. 3

    読取り，書込み，削除のすべて

正答と解説を確認

正答0. 読取りと削除

解説

左から読取り，書込み，削除を表すので，101は読取りが1で許可，書込みが0で不許可，削除が1で許可である。

QUESTION 134#center_style_curated_084_014

二つの正の8ビット2進数01111010₂と01001011₂を加算して得られる2進数として，最も適当なものはどれか。

  1. 0

    01100101₂

  2. 1

    01111111₂

  3. 2

    10000100₂

  4. 3

    11000101₂

正答と解説を確認

正答3. 11000101₂

解説

01111010₂は10進数で122，01001011₂は75である。122＋75＝197であり，197を8ビット2進数で表すと11000101₂になる。

QUESTION 135#center_style_curated_084_018

アプリケーションからの印刷要求を，OSを通じてプリンタ固有の制御命令へ変換するソフトウェアの役割として，最も適当なものはどれか。

  1. 0

    デバイスドライバが，OSと周辺機器の間を仲介する。

  2. 1

    コンパイラが，印刷要求を機械語の実行ファイルへ翻訳する。

  3. 2

    データベース管理システムが，印刷要求を表の行として保存する。

  4. 3

    Webサーバが，印刷要求をHTTPの応答として公開する。

正答と解説を確認

正答0. デバイスドライバが，OSと周辺機器の間を仲介する。

解説

デバイスドライバは，OSと周辺機器の間を仲介するソフトウェアである。OSから受け取った共通的な入出力要求を機器固有の制御命令へ変換し，プリンタやスキャナなどを利用できるようにする。機器の種類やOSに対応したドライバが必要になる。

QUESTION 136#center_style_curated_085_006

情報を連続的に変化する物理量で表したものをAデータ，離散的な数値で表したものをBデータという。また，音楽CDは情報をCデータとして記録する。A，B，Cの組合せとして，最も適当なものはどれか。

  1. 0

    アナログ，ディジタル，アナログ

  2. 1

    アナログ，ディジタル，ディジタル

  3. 2

    ディジタル，アナログ，アナログ

  4. 3

    ディジタル，アナログ，ディジタル

正答と解説を確認

正答1. アナログ，ディジタル，ディジタル

解説

連続的に変化する量で表すものはアナログデータ，離散的な数値で表すものはディジタルデータである。音楽CDは音をディジタルデータとして記録する。

QUESTION 138#center_style_curated_085_019

ファイルの読取り，書込み，実行の3種類の権限を3ビットで表す。試した結果，0ではどの権限もなく，3では読取りと書込みができ，7では読取り，書込み，実行がすべてできた。このとき，4を設定した場合として最も適当なものはどれか。

  1. 0

    読取りと実行ができる。

  2. 1

    実行だけができる。

  3. 2

    書込みだけができる。

  4. 3

    読取りと書込みができる。

正答と解説を確認

正答1. 実行だけができる。

解説

値0は000，値3は011，値7は111と考えられる。3で読取りと書込みができ，7でさらに実行が加わるので，4は実行だけを表すビットである。

QUESTION 139#original_knowledge_0023

100種類の状態を重複なく符号化するために必要な最小のビット数はどれか。

  1. 0

    6ビット

  2. 1

    8ビット

  3. 2

    7ビット

  4. 3

    100ビット

正答と解説を確認

正答2. 7ビット

解説

6ビットでは2⁶=64通りで不足し、7ビットなら2⁷=128通りを表せる。 bitは情報の最小単位であり、0であるか1であるかの2通りの情報を表す。 100通りの情報を表すには6bit＝64通りでは足らず、7bit＝128通りあればよい。 つまり、少なくとも7bit必要である。

QUESTION 140#original_knowledge_0024

2進数 110101₂ を10進数で表したものはどれか。

  1. 0

    45

  2. 1

    52

  3. 2

    61

  4. 3

    53

正答と解説を確認

正答3. 53

解説

1×2⁵+1×2⁴+1×2²+1×2⁰=32+16+4+1=53である。 2進数では右端から順に各桁が2⁰、2¹、2²、…の重みを持つ。110101₂では1の立っている桁が2⁵、2⁴、2²、2⁰なので、32+16+4+1=53となる。0の桁である2³と2¹は加算しない。
