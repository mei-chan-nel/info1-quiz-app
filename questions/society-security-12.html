QUESTION 111#center_style_fe_curated_122

持ち運び用PCを紛失したとき，保存されていたデータを読まれにくくする対策として最も適当なものはどれか。

  1. 0

    保存データを暗号化しておく。

  2. 1

    データのバックアップを別の場所に保管しておく。

  3. 2

    ファイル名を分かりにくい名前に変えておく。

  4. 3

    画面の明るさを低くしておく。

正答と解説を確認

正答0. 保存データを暗号化しておく。

解説

暗号化しておくと，PCを手に入れた第三者が保存データをそのまま読みにくくなる。

QUESTION 113#center_style_ip_curated_228

個人情報に当たらないものとして最も適当なものはどれか。

  1. 0

    氏名，住所，電話番号が載っている電話帳の情報

  2. 1

    自社の従業員の氏名と住所が載っている住所録

  3. 2

    年代別の人数だけを示し、個人を識別できる情報を含まない集計表

  4. 3

    顔がはっきり分かる防犯カメラの映像

正答と解説を確認

正答2. 年代別の人数だけを示し、個人を識別できる情報を含まない集計表

QUESTION 115#center_style_ip_curated_230

不正アクセスの例として最も適当なものはどれか。

  1. 0

    架空の商品を売ると言って代金を受け取る。

  2. 1

    他人のユーザIDとパスワードを無断で使い，他人の電子メールを読む。

  3. 2

    不正に複製した音楽CDを販売する。

  4. 3

    公開されている掲示板に，自分の意見を書き込む。

正答と解説を確認

正答1. 他人のユーザIDとパスワードを無断で使い，他人の電子メールを読む。

QUESTION 118#center_style_ap_curated_137

ソーシャルエンジニアリングの例として最も適当なものはどれか。

  1. 0

    緊急の連絡を装って電話し，パスワードや機密情報を聞き出そうとする。

  2. 1

    ファイルを暗号化して，第三者が内容を読みにくくする。

  3. 2

    重要なデータを失ったときに戻せるよう，バックアップを取る。

QUESTION 119#center_style_ip_curated_233

著作権が発生する時点として，最も適当なものはどれか。

  1. 0

    著作物を創作したとき

  2. 1

    著作物を他人に譲渡したとき

  3. 2

    著作物を複製したとき

  4. 3

    著作物を役所に登録したとき

正答と解説を確認

正答0. 著作物を創作したとき

解説

著作権は，著作物を創作した時点で発生する。日本では登録や届出をしなくても権利が生じるため，譲渡，複製，登録の時点ではない。

QUESTION 120#center_style_ip_curated_234

ディジタル署名を用いることで主に確認できることとして，最も適当なものはどれか。

  1. 0

    文書が改ざんされていないこと

  2. 1

    文書のデータ量が必ず小さくなること

  3. 2

    文書が必ず第三者に読まれないこと

  4. 3

    文書のバックアップが自動的に作られること

正答と解説を確認

正答0. 文書が改ざんされていないこと

解説

ディジタル署名は，署名者の確認や，署名後に文書が改ざんされていないことの確認に使われる。内容を秘密にする暗号化や，データ量を減らす圧縮とは目的が異なる。
