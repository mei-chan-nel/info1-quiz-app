QUESTION 021#center_style_common_test_0059

動画に関する次の記述a～cの正誤の組合せとして最も適当なものを，次の選択肢から一つ選べ。 a　動画共有サイトに動画をアップロードするとき，ストリーミングを利用すればデータの欠落を防止できる。 b　1920×1080で撮影した動画は，1920×1080のディスプレイでなければ再生できない。 c　各フレームを静止画と同様の方法で圧縮すれば，動画全体のデータ量を減らすことができる。

  1. 0

    a: 正、b: 正、c: 正

  2. 1

    a: 誤、b: 正、c: 誤

  3. 2

    a: 誤、b: 誤、c: 正

  4. 3

    a: 正、b: 誤、c: 正

正答と解説を確認

正答2. a: 誤、b: 誤、c: 正

解説

aは誤りである。ストリーミングは主に受信しながら再生する方式を指し，アップロード時のデータ欠落を防止する仕組みそのものではない。bも誤りである。1920×1080で撮影した動画は，拡大縮小して別の解像度のディスプレイでも再生できる。cは正しい。各フレームを静止画と同様に圧縮すれば，少なくともフレームごとのデータ量を減らせる。したがって「誤，誤，正」のこの内容が正答である。

QUESTION 022#center_style_fe_curated_001

60分のモノラル音声を、標本化周波数44.1kHz、量子化ビット数16ビットでディジタル化する。圧縮しない場合、データ量はおよそ何Mバイトか。ただし、1kHz＝1,000Hz、1Mバイト＝1,000,000バイトとする。

  1. 0

    80

  2. 1

    160

  3. 2

    320

  4. 3

    640

正答と解説を確認

正答2. 320

解説

1秒あたり44,100個の標本を16ビットで表すので、60分では44,100×16×3,600ビットである。8で割ると約317.5Mバイトとなり、最も近いのは320である。

QUESTION 023#center_style_fe_curated_002

8ビットのデータで、最上位ビットだけを0にし、下位7ビットは元のまま残したい。用いるビット演算として最も適当なものはどれか。

  1. 0

    16進数0FとのANDをとる

  2. 1

    16進数0FとのORをとる

  3. 2

    16進数7FとのANDをとる

  4. 3

    16進数FFとのXORをとる

正答と解説を確認

正答2. 16進数7FとのANDをとる

解説

16進数7Fは2進数で01111111である。これとANDをとると、最上位ビットは必ず0になり、下位7ビットは元の値が残る。

QUESTION 024#center_style_fe_curated_008

32ビットで表現できるビットパターンの個数は、24ビットで表現できる個数の何倍か。

  1. 0

    8

  2. 1

    16

  3. 2

    128

  4. 3

    256

正答と解説を確認

正答3. 256

解説

nビットで表せるパターン数は2ⁿ個である。32ビットと24ビットの差は8ビットなので、2³²÷2²⁴＝2⁸＝256倍となる。

QUESTION 025#center_style_fe_curated_009

A、B、Cが0または1をとる。出力が1になるのは、(A,B,C)が(0,0,1)、(1,0,1)、(0,1,1)、(1,1,1)のいずれかのときだけである。この出力と同じ値になるものとして最も適当なものはどれか。

  1. 0

    A

  2. 1

    B

  3. 2

    C

  4. 3

    A AND B

正答と解説を確認

正答2. C

解説

出力が1になる4通りでは、AとBは全ての組合せをとるが、Cは必ず1である。Cが0のときは出力が1にならないので、出力はCと同じである。

QUESTION 026#center_style_fe_curated_011

表示解像度が1,000×800ドットで、65,536色を表示する画像に必要なビデオメモリ容量は最低何Mバイトか。ここでは1Mバイト=1,000,000バイトとする。

  1. 0

    1.6

  2. 1

    3.2

  3. 2

    6.4

  4. 3

    12.8

正答と解説を確認

正答0. 1.6

解説

65,536色は2¹⁶色なので、1ドットに16ビット=2バイト必要である。1,000×800×2=1,600,000バイトで、1.6Mバイトとなる。

QUESTION 030#center_style_fe_curated_027

ファイルのアクセス権を3ビットで表し、読取り=4、書込み=2、実行=1として足し合わせる。値が5のとき許可されている操作として最も適当なものはどれか。

  1. 0

    読取りと実行

  2. 1

    書込みと実行

  3. 2

    読取りと書込み

  4. 3

    読取り、書込み、実行のすべて

正答と解説を確認

正答0. 読取りと実行

解説

読取り=4，書込み=2，実行=1を足し合わせて権限を表す。5は4+1なので，読取りと実行が許可され，書込みは含まれない。
