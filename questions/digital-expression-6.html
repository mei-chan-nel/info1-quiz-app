QUESTION 055#center_style_ap_curated_099

8ビットのD-A変換で，入力値0のとき出力が0V，入力値128のとき出力が2.5Vである。入力値が1増えたときの出力電圧の変化量として，最も適当なものはどれか。

  1. 0

    2.5/128 V

  2. 1

    2.5/255 V

  3. 2

    2.5/256 V

  4. 3

    2.5/512 V

正答と解説を確認

正答0. 2.5/128 V

解説

入力値0から128までの変化で出力電圧は0Vから2.5Vまで変化するので，入力値1当たりの変化量は2.5/128 Vである。問題文で与えられた対応範囲を使って比例計算する。

QUESTION 059#center_style_ap_curated_102

ランレングス圧縮は、同じ文字が連続する部分を「文字」と「連続する個数」の組で表す方法である。文字列「AAAAABB」をランレングス圧縮で表す。結果として、最も適当なものはどれか。

  1. 0

    A5B2

  2. 1

    A2B5

  3. 2

    AB7

  4. 3

    A4B3

正答と解説を確認

正答0. A5B2

解説

ランレングス圧縮では、同じ文字が続く長さを順に数えて表す。AAAAABBはAが5個連続し、その後Bが2個連続しているので、文字と連続個数で表すとA5B2になる。

QUESTION 060#center_style_ip_curated_143

10⁻⁹を表すSI接頭語と、10⁹を表すSI接頭語の記号の組合せとして、最も適当なものはどれか。

  1. 0

    n と G

  2. 1

    m と k

  3. 2

    p と M

  4. 3

    μ と T

正答と解説を確認

正答0. n と G

解説

SI接頭語では、10⁻⁹をナノといい記号はn、10⁹をギガといい記号はGで表す。小さい単位と大きい単位の向きを取り違えないようにする。
