QUESTION 031#center_style_ap_curated_021

マイクで入力した音声をコンピュータに保存し、保存した音声をスピーカで再生する。録音と再生で必要な変換の組合せとして最も適当なものはどれか。

  1. 0

    録音時はA/D変換、再生時はD/A変換を行う

  2. 1

    録音時はD/A変換、再生時はA/D変換を行う

  3. 2

    録音時は標本化だけを行えばよく、量子化や符号化は不要である

  4. 3

    録音と再生のどちらもD/A変換器だけで行う

正答と解説を確認

正答0. 録音時はA/D変換、再生時はD/A変換を行う

解説

マイクからの音声はアナログ信号なので、保存時にはA/D変換でディジタルデータにする。再生時には保存されたディジタルデータをD/A変換でアナログ信号に戻す。

QUESTION 032#center_style_ap_curated_056

音声を標本化周波数10kHz、量子化ビット数16ビットで4秒間サンプリングした。この音声データを4分の1に圧縮したとき、データ量は何kバイトか。ここで、1kバイトは1,000バイトとする。

  1. 0

    10

  2. 1

    20

  3. 2

    80

  4. 3

    160

正答と解説を確認

正答1. 20

解説

圧縮前は10,000回/秒×16ビット×4秒=640,000ビットで、80,000バイト=80kバイトである。4分の1に圧縮すると20kバイトになる。

QUESTION 037#center_style_fe_curated_058

1文字を縦48ドット、横32ドットの2値ビットマップで表すフォントがある。8,192種類の文字データを圧縮せずに保存するために必要な容量として最も適当なものはどれか。ここで、1Mバイト=1,024kバイト、1kバイト=1,024バイトとする。

  1. 0

    192kバイト

  2. 1

    1.5Mバイト

  3. 2

    12Mバイト

  4. 3

    96Mバイト

正答と解説を確認

正答1. 1.5Mバイト

解説

1文字は48×32=1,536ビットで、1,536÷8=192バイトである。8,192文字では192×8,192=1,572,864バイト=1.5Mバイトになる。

QUESTION 038#center_style_ip_curated_064

商品に三つの条件A、B、Cがある。AまたはBまたはCを満たす商品の件数を求めたい。Aだけで数えた件数をa、Bだけをb、Cだけをc、AかつBをab、AかつCをac、BかつCをbc、AかつBかつCをabcとすると、求める式として最も適当なものはどれか。

  1. 0

    a + b + c - ab - ac - bc + abc

  2. 1

    a + b + c + ab + ac + bc + abc

  3. 2

    ab + ac + bc - abc

  4. 3

    a + b + c - abc

正答と解説を確認

正答0. a + b + c - ab - ac - bc + abc

解説

A、B、Cのいずれかを満たす件数は包除原理で求める。a+b+cでは重なりを重複して数えるので、2条件の重なりを引き、3条件すべての重なりは引き過ぎた分として足し戻す。

QUESTION 039#center_style_ip_curated_090

表計算で、A2とB2に2科目の点数が入っている。合計が120点以上、かつ、2科目とも50点以上ならC2に「合格」、それ以外は「不合格」と表示したい。式として最も適当なものはどれか。

  1. 0

    =IF(AND(A2+B2>=120,A2>=50,B2>=50),"合格","不合格")

  2. 1

    =IF(AND(A2+B2>=120,A2>=50,B2>=50),"不合格","合格")

  3. 2

    =IF(OR(A2+B2>=120,A2>=50,B2>=50),"合格","不合格")

  4. 3

    =IF(OR(A2+B2>=120,A2>=50,B2>=50),"不合格","合格")

正答と解説を確認

正答0. =IF(AND(A2+B2>=120,A2>=50,B2>=50),"合格","不合格")

解説

三つの条件をすべて満たす必要があるのでANDを使い、真のときに「合格」、偽のときに「不合格」とする。

QUESTION 040#center_style_ip_curated_091

表計算で、A2とB2に2科目の点数が入っている。合計が120点以上、または、少なくとも一方が100点ならC2に「合格」、それ以外は「不合格」と表示したい。式として最も適当なものはどれか。

  1. 0

    =IF(AND(A2+B2>=120,A2=100,B2=100),"合格","不合格")

  2. 1

    =IF(AND(A2+B2>=120,A2=100,B2=100),"不合格","合格")

  3. 2

    =IF(OR(A2+B2>=120,A2=100,B2=100),"合格","不合格")

  4. 3

    =IF(OR(A2+B2>=120,A2=100,B2=100),"不合格","合格")

正答と解説を確認

正答2. =IF(OR(A2+B2>=120,A2=100,B2=100),"合格","不合格")

解説

合計が120点以上、A2が100点、B2が100点のうち一つでも満たせば合格なのでORを使い、真のときに「合格」とする。
