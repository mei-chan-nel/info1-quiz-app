QUESTION 021#center_style_ip_curated_104

Aさんは，BさんをTo，CさんをCc，DさんとEさんをBccに指定して電子メールを送った。このとき，受信者から見える宛先についての説明として最も適当なものはどれか。

  1. 0

    Bさんは，DさんとEさんがBccで受信したことを知ることができる。

  2. 1

    Cさんは，BさんがToで受信したことを知ることができる。

  3. 2

    Dさんは，EさんがBccで受信したことを知ることができる。

  4. 3

    Eさんは，DさんがBccで受信したことを知ることができる。

正答と解説を確認

正答1. Cさんは，BさんがToで受信したことを知ることができる。

解説

ToとCcの宛先は通常、受信者から見える。一方、Bccに指定された受信者はTo・Cc欄には表示されず、Bccで受信した人同士も通常は互いの指定を確認できない。したがって、CさんはBさんがToで受信したことを確認できる。

QUESTION 030#center_style_ap_curated_107

伝送速度2Mビット/秒の回線で50秒間データを送り、5ビットの誤りが発生した。この間のビット誤り率として最も適当なものはどれか。ただし、1Mビット＝1,000,000ビットとする。

  1. 0

    5.0×10⁻⁸

  2. 1

    2.0×10⁻⁷

  3. 2

    2.5×10⁻⁶

  4. 3

    5.0×10⁻⁵

正答と解説を確認

正答0. 5.0×10⁻⁸

解説

送信したデータは2,000,000×50＝100,000,000ビットである。5ビットの誤りが発生したので、ビット誤り率は5÷100,000,000＝5.0×10⁻⁸となる。
