QUESTION 079#center_style_curated_072_009

A-B間の回線の稼働率は0.9である。別経路としてA-C，C-Bの2本の回線を追加し，それぞれの稼働率も0.9である。AとBがつながる稼働率はどれか。

  1. 0

    0.729

  2. 1

    0.810

  3. 2

    0.981

  4. 3

    0.999

正答と解説を確認

正答2. 0.981

解説

別経路A-C-Bが使える確率は0.9×0.9=0.81である。直接経路も別経路も使えない確率は(1-0.9)×(1-0.81)=0.019なので，少なくとも一方が使える確率は1-0.019=0.981である。
