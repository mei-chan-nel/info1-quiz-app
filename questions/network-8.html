QUESTION 071#center_style_curated_069_004

無線LANで通信データを他人に読み取られにくくするために行う設定として，最も適当なものはどれか。

  1. 0

    通信データを暗号化する設定を行う。

  2. 1

    アンテナを2本に増やせば，暗号化は不要になる。

  3. 2

    接続する機器の画面の明るさを下げる。

  4. 3

    Webページの文字色だけを変える。

正答と解説を確認

正答0. 通信データを暗号化する設定を行う。

解説

無線LANでは電波を使うため，通信内容を読み取られにくくするには暗号化の設定が重要である。

QUESTION 072#center_style_curated_069_005

電子メールのBCC欄の説明として，最も適当なものはどれか。

  1. 0

    BCC欄の宛先は，To欄やCc欄の受信者には通知されない。

  2. 1

    BCC欄に入れると，自動的に添付ファイルが削除される。

  3. 2

    BCC欄に入れると，本文が必ず暗号化される。

  4. 3

    BCC欄に入れると，メールは送信されない。

正答と解説を確認

正答0. BCC欄の宛先は，To欄やCc欄の受信者には通知されない。

QUESTION 076#center_style_curated_072_009

A-B間の回線の稼働率は0.9である。別経路としてA-C，C-Bの2本の回線を追加し，それぞれの稼働率も0.9である。各回線の故障は互いに独立であるとき，AとBがつながる稼働率はどれか。

  1. 0

    0.729

  2. 1

    0.810

  3. 2

    0.981

  4. 3

    0.999

正答と解説を確認

正答2. 0.981

解説

別経路A-C-Bが使える確率は0.9×0.9＝0.81である。直接経路と別経路がともに使えない確率は，故障が互いに独立なので，(1－0.9)×(1－0.81)＝0.019となる。したがって，少なくとも一方が使える確率は1－0.019＝0.981である。

QUESTION 077#center_style_curated_072_010

A-B間の回線の稼働率は0.9，A-C間の回線の稼働率は0.8である。C-B間に稼働率pの予備回線を追加し，全体の稼働率を0.95以上にしたい。各回線の故障は互いに独立であるとき，pは最低いくら必要か。

  1. 0

    0.167

  2. 1

    0.205

  3. 2

    0.559

  4. 3

    0.625

正答と解説を確認

正答3. 0.625

解説

予備経路A-C-Bが使える確率は0.8pである。直接回線と各予備回線の故障が互いに独立なので，全体の稼働率は，直接回線が使える0.9に，直接回線が使えない0.1のとき予備経路が使える確率を加えた0.9＋0.1×0.8p＝0.9＋0.08pとなる。0.9＋0.08p≧0.95より，p≧0.625である。

QUESTION 080#center_style_curated_073_004
