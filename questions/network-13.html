QUESTION 121#simple_original_141

異なるメーカーの機器同士であっても通信できるように、データ形式や送受信手順などを定めた取り決めはどれか。

  1. 0

    インタフェース

  2. 1

    トポロジ

  3. 2

    アドレス

  4. 3

    プロトコル

正答と解説を確認

正答3. プロトコル

解説

通信で用いるデータ形式、手順、誤り処理などを定めた規約をプロトコルという。同じメーカーの機器同士でも共通のプロトコルに従う必要がある。 メーカーが同じか異なるかにかかわらず、通信する双方が共通のプロトコルに従う必要がある。プロトコルによりデータ形式や応答手順を解釈できる。

QUESTION 122#simple_original_142

インターネットで受け取ったパケットについて、宛先IPアドレスなどを基に次に送る方向を判断する機器はどれか。

  1. 0

    ルータ

  2. 1

    ハブ

  3. 2

    Webブラウザ

  4. 3

    SMTPサーバ

正答と解説を確認

正答0. ルータ

解説

ルータは経路制御表などを使い、パケットを適切な次のネットワークへ転送する。 ルータは宛先IPアドレスと経路情報を基に、パケットを次のネットワークへ転送する。ハブは同じLAN内の接続、ブラウザはWeb閲覧を担う。

  1. 0

    LAN

  2. 1

    WAN

  3. 2

    PAN

  4. 3

    USB

正答と解説を確認

正答1. WAN

解説

WANは、地理的に離れた拠点やLAN同士を結ぶ広域ネットワークである。 WANは地理的に離れた拠点やLAN同士を回線で結ぶ広域ネットワークである。LANは比較的狭い範囲、PANは個人の周辺、USBは接続規格を指す。

QUESTION 124#simple_original_144

WebサーバがWebページを提供し、利用者のブラウザが要求して表示する構成はどれか。

  1. 0

    P2P方式

  2. 1

    スタンドアロン方式

  3. 2

    クライアントサーバ方式

  4. 3

    バッチ処理

正答と解説を確認

正答2. クライアントサーバ方式

解説

クライアントサーバ方式では、サービスを提供するサーバと、それを利用するクライアントに役割を分ける。 クライアントサーバ方式では、Webサーバがサービスを提供し、ブラウザなどのクライアントが要求して利用する。P2P方式では端末同士が対等に資源を扱う。

QUESTION 126#simple_original_147

インターネットで使われている代表的なプロトコル群はどれか。

  1. 0

    HTML/CSS

  2. 1

    TCP/IP

  3. 2

    RGB/CMY

  4. 3

    SQL/DBMS

正答と解説を確認

正答1. TCP/IP

解説

TCP/IPはインターネットで広く用いられるプロトコル群である。IPによる配送やTCPによる通信制御など、複数のプロトコルを含む。 HTML/CSSはWeb文書、RGB/CMYは色、SQL/DBMSはデータベースに関する用語の組合せである。インターネット通信の基盤となるプロトコル群はTCP/IPである。

QUESTION 129#simple_original_151

校内ネットワークの境界で、規則に基づき外部からの通信を許可又は遮断する仕組みはどれか。

  3. 2
