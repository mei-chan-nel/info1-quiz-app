QUESTION 011#center_style_ip_curated_028

学生表、科目表、成績表があり、成績表には存在する学生と科目だけを登録したい。このとき成績表に設定する外部キーの考え方として最も適当なものはどれか。

  1. 0

    成績表の学生番号は学生表の学生番号を参照し、成績表の科目コードは科目表の科目コードを参照する

  2. 1

    学生表の学生名は科目表の科目名を参照する

  3. 2

    成績表の成績欄を主キーとして、学生表と科目表のすべての列を参照する

  4. 3

    外部キーを設定すると、存在しない学生番号や科目コードも自由に登録できる

正答と解説を確認

正答0. 成績表の学生番号は学生表の学生番号を参照し、成績表の科目コードは科目表の科目コードを参照する

解説

外部キーは、他の表の主キーなどを参照して、存在しない値が登録されることを防ぐために使う。

QUESTION 016#center_style_ap_curated_027

データベースのフルバックアップを取る間隔を、これまでの2倍にした。更新の頻度がほぼ一定なら、障害時の復旧について起こりやすいこととして最も適当なものはどれか。

  1. 0

    ログを使って反映する更新量が増え、復旧処理時間が平均して長くなる

  2. 1

    1回のフルバックアップに必要な容量が必ず2倍になる

  3. 2

    1回のフルバックアップに必要な容量が必ず半分になる

  4. 3

    フルバックアップを取る1回当たりの処理時間が必ず2倍になる

正答と解説を確認

正答0. ログを使って反映する更新量が増え、復旧処理時間が平均して長くなる

解説

フルバックアップの間隔を長くすると、バックアップ後にログから反映しなければならない更新が増えるため、復旧処理時間は平均して長くなりやすい。

QUESTION 019#center_style_ip_curated_048

表計算ソフトで、セルA2に買い物金額が入っている。1,000円以下は1%、1,000円を超え3,000円以下は2%、3,000円を超える場合は3%のポイントを小数点以下切捨てで求める式として最も適当なものはどれか。

  1. 0

    IF(A2<=1000, 整数部(A2*1/100), IF(A2<=3000, 整数部(A2*2/100), 整数部(A2*3/100)))

  2. 1

    IF(A2<=1000, 整数部(A2*3/100), IF(A2<=3000, 整数部(A2*2/100), 整数部(A2*1/100)))

  3. 2

    IF(A2<3000, 整数部(A2*3/100), 整数部(A2*1/100))

  4. 3

    整数部(A2*3/100)

正答と解説を確認

正答0. IF(A2<=1000, 整数部(A2*1/100), IF(A2<=3000, 整数部(A2*2/100), 整数部(A2*3/100)))

解説

条件は金額の小さい順に判定すればよい。A2<=1000なら1%、そうでなくA2<=3000なら2%、それ以外は3%を掛け、小数点以下を切り捨てる。

QUESTION 020#center_style_ip_curated_051

データ 10, 20, 20, 26, 40, 50, 100, 440, 2000 の平均値と中央値の組合せとして最も適当なものはどれか。

  1. 0

    平均値 40、中央値 40

  2. 1

    平均値 約300.7、中央値 50

  3. 2

    平均値 約300.7、中央値 40

  4. 3

    平均値 2706、中央値 40

正答と解説を確認

正答2. 平均値 約300.7、中央値 40

解説

合計は2706で、データ数は9個なので平均値は2706÷9=約300.7である。小さい順に並んだ9個の中央、5番目は40なので中央値は40である。
