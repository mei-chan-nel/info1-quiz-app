QUESTION 061#center_style_ap_curated_052

生体認証の特徴として最も適当なものはどれか。

  1. 0

    指紋や顔など，本人の身体的な特徴を利用して本人確認を行う。

  2. 1

    全員が同じパスワードを入力すれば安全性が高くなる。

  3. 2

    認証に使う情報は，他人に知られても変更せずに使い続けられる。

  4. 3

    バックアップを取るための方法であり，本人確認には関係しない。

正答と解説を確認

正答0. 指紋や顔など，本人の身体的な特徴を利用して本人確認を行う。

解説

生体認証は，指紋や顔など本人の身体的な特徴を使って本人確認を行う方法である。

QUESTION 062#center_style_ip_curated_056

インターネットカフェなどの共用PCでWebサイトを利用した後、Cookieについて行うべき対応として最も適当なものはどれか。

  1. 0

    Cookieには個人に関する情報が含まれる場合があるので、利用後に削除する

  2. 1

    Cookieは別のPCにも自動で引き継がれるので、削除しても意味がない

  3. 2

    Cookieは画像ファイルだけを保存するので、個人情報とは関係がない

  4. 3

    Cookieを保存しておけば、次の利用者にも自分のログイン状態を使わせられて安全である

正答と解説を確認

正答0. Cookieには個人に関する情報が含まれる場合があるので、利用後に削除する

解説

Cookieにはログイン状態や利用履歴など、個人に関わる情報が含まれる場合がある。共用PCでは利用後に削除するのが望ましい。

QUESTION 065#center_style_ip_curated_061

小文字のアルファベット a から z に，順に 0 から 25 を対応させる。小文字1文字を番号に直し，5を加えて26で割った余りの番号の文字に置き換える。文字 x をこの規則で置き換えた結果として最も適当なものはどれか。

  1. 0

    a

  2. 1

    b

  3. 2

    c

  4. 3

    d

正答と解説を確認

正答2. c

解説

xは23なので、23+5=28である。28を26で割った余りは2で、2に対応する文字はcである。

QUESTION 068#center_style_fe_curated_061

アプリケーションソフトウェアにディジタル署名を付ける主な目的として最も適当なものはどれか。

  1. 0

    利用者がソフトウェアの改ざんを検知できるようにする

  2. 1

    利用者による画面の拡大縮小を不可能にする

  3. 2

    ソフトウェアのファイルサイズを必ず小さくする

  4. 3

    ソフトウェアを使う時間を自動的に短くする

正答と解説を確認

QUESTION 069#center_style_sg_curated_031

著作者人格権に含まれる権利の例として最も適当なものはどれか。

  1. 0

    著作物を印刷，撮影，複写などの方法で複製する権利

  2. 1

    公衆からの要求に応じてサーバから著作物を送信できるようにする権利

  3. 2

    著作物の複製物を公衆に貸し出す権利

  4. 3

    自分の意思に反して著作物を改変されない権利

正答と解説を確認

正答3. 自分の意思に反して著作物を改変されない権利

解説

著作者人格権は，著作者の人格的な利益を守る権利であり，同一性保持権は自分の意思に反して著作物を改変されない権利である。

QUESTION 070#center_style_ip_curated_102

著作物の利用例として，著作権を侵害するおそれが最も高いものはどれか。

  1. 0

    放送された音楽番組を家庭内で視聴するために録画する。

  2. 1

    他人が公開した文章を無断で翻訳し，自分のWebサイトに掲載する。

  3. 2

    研究論文の一部を，引用箇所と出所を明示して必要な範囲で引用する。

  4. 3

    利用条件で再利用が許されている資料を，その条件に従って使う。

正答と解説を確認

正答1. 他人が公開した文章を無断で翻訳し，自分のWebサイトに掲載する。

解説

他人の文章を翻訳してWebサイトで公開することは，翻訳や公衆送信に関わる利用であり，許諾なしでは著作権侵害のおそれが高い。
